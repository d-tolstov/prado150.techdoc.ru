DTC    C1611    Неисправность ЭБУ

для подготовки Нажмите здесь


ОПИСАНИЕ

Данный DTC сохраняется, если ЭБУ системы помощи при парковке в результате самодиагностики обнаруживает внутреннюю неисправность.

УКАЗАНИЕ:
Во время инициализации ЭБУ системы помощи при парковке сохраняет информацию различных типов. Если при включении ЭБУ системы помощи при парковке не может считать сохраненную информацию, он регистрирует внутреннюю неисправность и сохраняет данный DTC.


Код DTC
Условие обнаружения DTC
Неисправный участок
C1611
  1. EEPROM Status (Initial read)
  2. AISIC Output Status
  1. Инициализация ЭБУ системы помощи при парковке (неинициализированный ЭБУ системы помощи при парковке)
  2. Настройка положения камеры заднего вида (инициализированный ЭБУ системы помощи при парковке)
  3. Настройка положения боковой камеры (инициализированный ЭБУ системы помощи при парковке)*
  4. Установка угла поворота рулевого колеса (инициализированный ЭБУ системы помощи при парковке)
  5. ЭБУ системы помощи при парковке
*: Для моделей с монитором бокового обзора

ПОСЛЕДОВАТЕЛЬНОСТЬ ПРОВЕРКИ

ПРИМЕЧАНИЕ:
  1. Если после отсоединения вывода аккумуляторной батареи на дисплее модуля системы навигации и индикации отображается сообщение "System initializing" (инициализация системы), отрегулируйте нейтральное положение рулевого колеса (Нажмите здесь).
  2. В зависимости от замененных деталей или операций, которые были выполнены при проверке или обслуживании автомобиля, может потребоваться калибровка других систем, а также системы помощи при парковке (Нажмите здесь).

2.УСТАНОВКИ (ИНИЦИАЛИЗАЦИЯ ЭБУ СИСТЕМЫ ПОМОЩИ ПРИ ПАРКОВКЕ)

УКАЗАНИЕ:
  1. Во время инициализации ЭБУ системы помощи при парковке сохраняет информацию различных типов. Если при включении ЭБУ системы помощи при парковке не может считать сохраненную информацию, он регистрирует внутреннюю неисправность и сохраняет данный DTC.
  2. Если при включении ЭБУ системы помощи при парковке не может считать сохраненную информацию, нормальная работа системы может восстановиться, если дать ему сохранить информацию еще раз.

  1. Перейдите на экраны диагностики (обратите внимание на наблюдаемую схему переходов), чтобы убедиться, что ЭБУ системы помощи при парковке может правильно сохранять данные инициализации (Нажмите здесь).

    УКАЗАНИЕ:
    Процедура сохранения информации об автомобиле в ЭБУ системы помощи при парковке различается в зависимости от того, есть ли в нем сохраненная информация (ЭБУ инициализирован), либо этой информации нет (ЭБУ не инициализирован).

  1. Выполните следующую процедуру в зависимости от состояния памяти, чтобы зарегистрировать информацию об автомобиле в ЭБУ системы помощи при парковке.

    Условие
    		Порядок выполнения
    Не инициализирован
    		Инициализация ЭБУ системы помощи при парковке (Нажмите здесь)
    Инициализирован
    1. Настройка положения задней камеры (Нажмите здесь)
    2. Настройка угла поворота рулевого колеса (система диагностики) (Нажмите здесь)
    3. Настройка положения боковой камеры (Нажмите здесь)

УКАЗАНИЕ:
  1. Если выводится DTC C1611, снова выполните процедуру, чтобы ЭБУ системы помощи при парковке смог сохранить информацию об автомобиле.
  2. Если после 3-кратного выполнения процедуры сохранения информации об автомобиле в ЭБУ системы помощи при парковке по-прежнему выводится DTC C1611, замените ЭБУ системы помощи при парковке.
  3. Если DTC C1611 выводится часто, замените ЭБУ системы помощи при парковке, даже если при повторной проверке DTC коды не выводятся.
