ОПИСАНИЕ

  1. Когда ECM регистрирует сигнал включения выключателя круиз-контроля, он направляет его через шину CAN в щиток приборов. Затем включается контрольная лампа круиз-контроля SET.
  2. Для обмена данными с цепью контрольной лампы круиз-контроля SET используется шина CAN. Если в данной цепи имеется неисправность, перед поиском неисправности проверьте, не выводятся ли коды DTC, относящиеся к системе передачи данных CAN.
