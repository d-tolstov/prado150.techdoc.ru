НАСТРОЙКА ПАРАМЕТРОВ С ПОМОЩЬЮ ПОРТАТИВНОГО ДИАГНОСТИЧЕСКОГО ПРИБОРА

ПРИМЕЧАНИЕ:
  1. Если клиент требует изменить настройки, прежде всего следует убедиться, что значения соответствующих параметров могут изменяться.
  2. Прежде чем приступать к настройке параметров, запишите текущие установки.

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Установите замок зажигания в положение ON (ВКЛ).

  3. Включите портативный диагностический прибор.

  4. Войдите в следующие меню: Utility / Customize Setting.

  5. Выберите установки в соответствии со следующей таблицей.

    Slide Roof 
    Дисплей
    		Значение по умолчанию
    		Содержание
    		Настройка
    Door Key Related Open*1 (открывание)
    		ON (ВКЛ)
    		Функция, которая позволяет открывать люк вручную одновременно с окнами путем удержания ключа в двери водителя в положении разблокировки не менее 2,0 с при выключенном зажигании.
    		ON (ВКЛ) или OFF (ВЫКЛ)
    Door Key Related Close*1 (закрывание с помощью ключа двери)
    		ON (ВКЛ)
    		Функция, которая позволяет закрывать люк вручную одновременно с окнами путем удержания ключа в двери водителя в положении блокировки не менее 2,0 с при выключенном зажигании.
    		ON (ВКЛ) или OFF (ВЫКЛ)
    Wireless Key Related Open*2 (открывание с помощью передатчика)
    		ON (ВКЛ)
    		Функция, которая позволяет открывать люк вручную одновременно с окнами путем нажатия переключателя UNLOCK передатчика не менее 3,0 с при выключенном зажигании.
    		ON (ВКЛ) или OFF (ВЫКЛ)
    Wireless Key Related Close*2 (открывание с помощью передатчика)
    		ON (ВКЛ)
    		Функция, которая позволяет закрывать люк вручную одновременно с окнами путем нажатия переключателя LOCK передатчика не менее 3,0 с при выключенном зажигании.
    		ON (ВКЛ) или OFF (ВЫКЛ)
    Door Key Related Operation
    		Slide
    		Функция, которая позволяет вручную поднимать люк или открывать его сдвигом одновременно с окнами путем удержания ключа в двери водителя в положении разблокировки не менее 2,0 с при выключенном зажигании.
    		Slide или Tilt
    Wireless Key Related Operation (открывание или закрывание с помощью передатчика)
    		Slide
    		Функция, которая позволяет вручную поднимать люк или открывать его сдвигом одновременно с окнами путем нажатия переключателя UNLOCK передатчика не менее 3,0 с при выключенном зажигании.
    		Slide или Tilt

    УКАЗАНИЕ:
    1. *1: Данная функция связана с функцией управления стеклоподъемниками с помощью ключа. Если функция управления стеклоподъемниками с помощью ключа отключена, то функция управления люком с помощью ключа не будет работать, даже если она включена.
    2. *2: Данная функция связана с функцией дистанционного управления стеклоподъемниками с помощью передатчика. Если функция управления стеклоподъемниками с помощью передатчика отключена, то функция управления люком с помощью передатчика не будет работать, даже если она включена.
