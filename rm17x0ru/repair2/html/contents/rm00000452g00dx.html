DTC    C1379    Неисправность переключателя системы помощи при спуске по склону (DTC режима активной диагностики)

для подготовки Нажмите здесь


ОПИСАНИЕ

Для моделей с системой помощи при спуске по склону:
При выходе из режима активной диагностики или передаче переключателем системы помощи при спуске по склону сигнала включения системы помощи при спуске по склону DTC C1379 удаляется.
Для моделей с системой Crawl:
При выходе из режима активной диагностики или передаче переключателем системы Crawl сигнала включения системы Crawl DTC C1379 удаляется.

Код DTC
Условие обнаружения DTC
Неисправный участок
C1379
Сохраняется только в режиме активной диагностики.
  1. Переключатель системы помощи при спуске по склону (для моделей с системой помощи при спуске по склону)
  2. Переключатель Crawl (для моделей с системой Crawl)
  3. Жгут или разъем
  4. Электромагнитный клапан главного цилиндра (ЭБУ системы противоскольжения)
