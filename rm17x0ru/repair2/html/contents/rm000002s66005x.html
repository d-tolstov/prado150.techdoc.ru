ИЛЛЮСТРАЦИЯ СИСТЕМЫ УПРАВЛЕНИЯ ВПУСКОМ ВОЗДУХА
На впускном коллекторе смонтирован клапан регулирование всасывания. Электровакуумный клапан (VSV) для клапана регулирования всасывания изменяет разрежение для управления приводом.
Затем он с помощью VSV клапана регулирования всасывания изменяет разрежение, прикладываемое к мембране привода, тем самым открывая и закрывая клапан регулирования всасывания.

СХЕМА СОЕДИНЕНИЙ СИСТЕМЫ УПРАВЛЕНИЯ ВПУСКОМ ВОЗДУХА





ИЛЛЮСТРАЦИЯ СИСТЕМЫ ТУРБОНАГНЕТАТЕЛЯ
  1. В систему турбонагнетателя входят собственно турбонагнетатель с регулируемыми соплами (типа VN) и ECM.
  2. Турбонагнетатель имеет лопасти сопла, которые открываются и закрываются для регулирования объема отработавших газов, поступающего в турбину. Благодаря этому регулируется давление наддува. Когда лопасти сопла двигаются по направлению закрывания, давление повышается. Когда лопасти двигаются по направлению открывания, давление понижается.
  3. Лопатки сопла приводит в движение привод турбонагнетателя со стороны кожуха компрессора. Датчик положения сопловой лопатки на приводе определяет угол поворота лопатки сопла. ECM получает сигнал датчика положения сопловой лопатки. На основании этого сигнала ECM приводит в движение электродвигатель постоянного тока и регулирует угол открывания турбонагнетателя.
  4. ECM осуществляет управление с целью достижения положения лопаток сопла, отвечающего оптимальному давлению, в соответствии с условиями движения.
