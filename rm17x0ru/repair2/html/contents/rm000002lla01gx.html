ОПИСАНИЕ

  1. Данные и диагностические коды неисправностей (DTC) системы кондиционирования можно считывать через разъем шины данных 3 (DLC3) автомобиля. Если предполагается, что система неисправна, произведите диагностику с помощью портативного диагностического прибора.


ПРОВЕРЬТЕ DLC3

  1. Проверьте DLC3 (Нажмите здесь).


ОПИСАНИЕ МЕТОДОВ ДЕЙСТВИЙ


  1. Для входа в режим активной диагностики установите переключатели системы кондиционирования в положения, показанные на приведенной ниже схеме.




ПРОВЕРКА ИНДИКАЦИИ

  1. Выключите зажигание.

  2. Установите замок зажигания в положение ACC и подождите не менее 5 с.

  3. Одновременно нажмите и удерживайте переключатель "AUTO" и переключатель рециркуляции/забора свежего воздуха системы кондиционирования, а затем включите зажигание. Продолжайте удерживать 2 переключателя, пока не активируется режим диагностики панели.




  4. Убедитесь, что все индикаторы загораются и гаснут последовательно 4 раза с интервалом в 1 секунду.

    УКАЗАНИЕ:
    1. После завершения проверки индикаторов система автоматически перейдет в режим проверки DTC.
    2. Нажмите на переключатель "OFF", чтобы выйти из режима проверки.


ПРОВЕРКА DTC (ПРОВЕРКА ДАТЧИКОВ)

  1. Запустите и прогрейте двигатель.

  2. Выполните проверку индикации.

    УКАЗАНИЕ:
    После завершения проверки индикаторов система автоматически перейдет в режим проверки DTC.

  3. Считайте код, отображаемый на панели.

    ПРИМЕЧАНИЕ:
    В режиме проверки датчиков, который автоматически вступает в действие после проверки индикаторов, поиск неисправностей может быть выполнен только частично.

    УКАЗАНИЕ:
    Описание кодов см. в таблице диагностических кодов неисправностей (Нажмите здесь).
    1. В отсутствие неисправностей выводится код DTC 00.
    2. В качестве примера на рисунке показан экран при выводе кода DTC 23.

  4. Если чтение кодов затруднено из-за их автоматического изменения, нажмите переключатель повышения температуры со стороны водителя, чтобы коды отображались поочередно для облегчения их чтения. При каждом нажатии переключателя повышения температуры со стороны водителя на дисплее будет отображаться очередной код.

    УКАЗАНИЕ:
    Нажмите на переключатель OFF (ВЫКЛ), чтобы завершить диагностику с панели.
