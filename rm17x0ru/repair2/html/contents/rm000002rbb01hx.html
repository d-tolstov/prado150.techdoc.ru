ОПИСАНИЕ

Когда дверь разблокируется, ЭБУ сертификации передает сигнал разблокировки двери и сигнал идентификационного кода ключа в главный ЭБУ кузова (бортовой ЭБУ сети мультиплексной связи). Когда главный ЭБУ кузова (бортовой ЭБУ сети мультиплексной связи) получает сигнал разблокировки двери, он сохраняет идентификационный код ключа. При открывании двери водителя во все ЭБУ, участвующие в регулировке положения для вождения, передаются данные для обращения к памяти, соответствующие сохраненному идентификационному коду ключа. Каждый ЭБУ управления получает команду обращения к памяти и осуществляет восстановление из памяти в соответствии с заданным временем выполнения операции.
