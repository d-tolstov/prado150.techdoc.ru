1. INSTALL LEVEL WARNING SWITCH

  1. Install a new level warning switch gasket to the level warning switch.

  1. Apply fuel to the level warning switch gasket of the level warning switch.




  1. Install the level warning switch to the fuel filter element and tighten it by hand.
    Text in Illustration 
    *a
    		Front
    *b
    		Within 28.4°

    УКАЗАНИЕ:
    Install the level warning switch within the angle range shown in the illustration.



2. INSTALL FUEL FILTER CASE

  1. Install the fuel filter gasket and fuel filter case to the fuel filter element.




  1. Tighten the 2 bolts.

    Момент затяжки:
    3.4 N*m{ 35 kgf*cm , 30 in.*lbf }

    УКАЗАНИЕ:
    Install the level warning switch within the angle range shown in the illustration.

  1. Attach the level warning switch connector clamp to the fuel filter cap.
    Text in Illustration 
    *a
    		Front
    *b
    		0 to 20°



3. INSTALL FUEL FILTER ASSEMBLY

  1. Install the fuel filter with the 2 nuts.

    Момент затяжки:
    18 N*m{ 184 kgf*cm , 13 ft.*lbf }




  1. Connect the level warning switch connector.

  1. Connect the 2 fuel hoses.
    Text in Illustration 
    *a
    		Upper



4. BLEED AIR FROM FUEL SYSTEM



  1. Using the hand pump mounted on the fuel filter cap, bleed the air from the fuel system. Continue pumping until the pump resistance increases.



5. INSPECT FOR FUEL LEAK

  1. Check that there are no fuel leaks anywhere in the fuel system after performing maintenance.

    УКАЗАНИЕ:
    When checking for fuel leaks, make sure that there is pressure in the fuel line.
