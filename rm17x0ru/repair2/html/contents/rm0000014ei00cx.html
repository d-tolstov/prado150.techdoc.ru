1. INSTALL STARTER ARMATURE ASSEMBLY



  1. Using a press, install the starter armature assembly.



2. INSTALL STARTER BRUSH HOLDER ASSEMBLY



  1. Align the claw of the brush holder with the claw groove of the starter yoke.
    Text in Illustration 
    *1
    		Claw
    *2
    		Groove

  1. Place the brush holder on the starter yoke.

  1. Connect the 4 brushes to the starter brush holder.

    1. Using a screwdriver, hold back the brush spring.

    2. Connect the brush to the starter brush holder.

      ПРИМЕЧАНИЕ:
      Check that the positive (+) lead wires are not grounded.



3. INSTALL MAGNET STARTER SWITCH ASSEMBLY

  1. Apply high-temperature grease to the starter idle pinion, starter clutch, starter armature pinion, retainer and clutch roller.




  1. Insert the steel ball into the starter clutch hole.

  1. Insert the return spring into the starter clutch hole.

  1. Install the starter clutch, idle gear, retainer and clutch roller to the starter drive housing.

  1. Install the starter housing to the magnet starter switch with the 2 screws.

    Момент затяжки:
    9.3 N*m{ 95 kgf*cm , 82 in.*lbf }



4. INSTALL STARTER YOKE ASSEMBLY

  1. Install a new O-ring to the groove of the starter yoke.




  1. Align the claw of the starter brush holder with the grooves of the magnet starter switch.
    Text in Illustration 
    *1
    		Claw
    *2
    		Groove

  1. Install the 2 new O-rings to the bolts.

  1. Install the 2 bolts.

    Момент затяжки:
    13 N*m{ 130 kgf*cm , 9 ft.*lbf }

  1. Connect the terminal C wire with the nut.

    Момент затяжки:
    5.9 N*m{ 60 kgf*cm , 52 in.*lbf }

  1. Install the terminal 50 wire with the screw.

    Момент затяжки:
    3.6 N*m{ 37 kgf*cm , 32 in.*lbf }
