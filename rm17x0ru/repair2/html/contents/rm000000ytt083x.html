ОПИСАНИЕ

Цепь пиропатрона подушки безопасности для защиты коленей водителя включает центральный блок управления системы SRS и подушку безопасности для защиты ног № 1.
Данная цепь служит для передачи управляющего сигнала системы SRS о необходимости активации пиропатрона при выполнении соответствующих условий.
Данные коды DTC регистрируются при обнаружении неисправности в цепи пиропатрона подушки безопасности для защиты коленей водителя.

Код DTC
Условие обнаружения DTC
Неисправный участок
B1860/64
Выполняется одно из следующих условий:
  1. Центральный блок управления системы SRS обнаруживает сигнал короткого замыкания в цепи пиропатрона подушки безопасности для защиты коленей водителя 5 раз в ходе выполнения первичной проверки.
  2. Неисправность пиропатрона подушки безопасности для защиты коленей водителя.
  3. Неисправность центрального блока управления системы SRS.
  1. Жгут проводов панели приборов
  2. Подушка безопасности для защиты ног № 1 в сборе (пиропатрон подушки безопасности для защиты коленей водителя)
  3. Центральный блок управления системы SRS
B1861/64
Выполняется одно из следующих условий:
  1. Центральный блок управления системы SRS обнаруживает сигнал обрыва в цепи пиропатрона подушки безопасности для защиты коленей водителя в течение 2 с.
  2. Неисправность пиропатрона подушки безопасности для защиты коленей водителя.
  3. Неисправность центрального блока управления системы SRS.
  1. Жгут проводов панели приборов
  2. Подушка безопасности для защиты ног № 1 в сборе (пиропатрон подушки безопасности для защиты коленей водителя)
  3. Центральный блок управления системы SRS
B1862/64
Выполняется одно из следующих условий:
  1. Центральный блок управления системы SRS обнаруживает сигнал короткого замыкания на массу в цепи пиропатрона подушки безопасности для защиты коленей водителя в течение 0,5 с.
  2. Неисправность пиропатрона подушки безопасности для защиты коленей водителя.
  3. Неисправность центрального блока управления системы SRS.
  1. Жгут проводов панели приборов
  2. Подушка безопасности для защиты ног № 1 в сборе (пиропатрон подушки безопасности для защиты коленей водителя)
  3. Центральный блок управления системы SRS
B1863/64
Выполняется одно из следующих условий:
  1. Центральный блок управления системы SRS обнаруживает сигнал короткого замыкания на B+ в цепи пиропатрона подушки безопасности для защиты коленей водителя в течение 0,5 с.
  2. Неисправность пиропатрона подушки безопасности для защиты коленей водителя.
  3. Неисправность центрального блока управления системы SRS.
  1. Жгут проводов панели приборов
  2. Подушка безопасности для защиты ног № 1 в сборе (пиропатрон подушки безопасности для защиты коленей водителя)
  3. Центральный блок управления системы SRS
