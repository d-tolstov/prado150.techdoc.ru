ОПИСАНИЕ

Данная цепь обеспечивает питание, необходимое для работы управляющего реле очистителя передних фар.
Реле управления очистителем фар передает сигнал выключателя очистителя фар в сборе и прочие сигналы в электродвигатель очистителя фар и насос в сборе для приведения в действие системы очистителей фар.

СХЕМА СОЕДИНЕНИЙ




ПОСЛЕДОВАТЕЛЬНОСТЬ ПРОВЕРКИ

ПРИМЕЧАНИЕ:
Перед выполнением следующей процедуры проверки проверьте предохранители цепей, относящихся к данной системе.

1.ВЫПОЛНИТЕ АКТИВНУЮ ДИАГНОСТИКУ С ПОМОЩЬЮ ПОРТАТИВНОГО ДИАГНОСТИЧЕСКОГО ПРИБОРА (РАБОТА ОЧИСТИТЕЛЯ ФАР)

  1. Выполните диагностику в режиме Active Test с помощью портативного диагностического прибора (Нажмите здесь).

    Main Body (Кузов) 
    Информация на дисплее прибора
    		Испытываемое устройство
    		Диапазон регулирования
    		Замечание по диагностике
    Head Light Cleaner
    		Работа очистителя фар
    		ON (ВКЛ) / OFF (ВЫКЛ)
    		-

    УКАЗАНИЕ:
    Система очистителей фар приводится в действие только при первом включении выключателя омывателя переднего стеклоочистителя (переключателя стеклоочистителя ветрового стекла). После этого система очистителей фар не сможет работать с управлением от выключателя омывателя переднего стеклоочистителя (переключателя стеклоочистителя ветрового стекла), пока зажигание не будет выключено, а потом включено снова.

    OK:
