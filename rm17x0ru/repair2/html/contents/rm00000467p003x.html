ПРЕДОСТЕРЕЖЕНИЕ:
Работайте в защитных перчатках. Острые края деталей могут травмировать руки.

УКАЗАНИЕ:
  1. Порядок выполнения работ одинаков для моделей с левосторонним и правосторонним рулевым управлением.
  2. Ниже рассмотрен порядок выполнения работ для моделей с левосторонним рулевым управлением.
  3. Порядок выполнения работ для левой и правой стороны одинаков.
  4. Ниже рассмотрен порядок выполнения работ для левой стороны.



1. УСТАНОВИТЕ ПОДОГРЕВАТЕЛЬ СПИНКИ ЛЕВОГО ПЕРЕДНЕГО СИДЕНЬЯ В СБОРЕ



  1. Установите подогреватель спинки сиденья, обратив сторону со штампом наименования к обивке спинки сиденья.
    Обозначения на рисунке 
    *1
    		Стяжка
    *2
    		Обивка спинки сиденья
    *3
    		Подогреватель спинки сиденья
    *4
    		Штамп наименования

  1. Закрепите подогреватель спинки сиденья новыми закрепками.



2. УСТАНОВИТЕ ОБИВКУ СПИНКИ РАЗДЕЛЬНОГО ПЕРЕДНЕГО СИДЕНЬЯ


  1. Установите обивку подушки сиденья на подушку сиденья и закрепите ее новыми витковыми пружинами, используя щипцы для витковых пружин.

    ПРИМЕЧАНИЕ:
    1. Соблюдайте осторожность, чтобы не повредить обивку.
    2. При установке витковых пружин будьте аккуратны, старайтесь избегать образования складок на обивке.

    УКАЗАНИЕ:
    Количество витковых пружин различается в зависимости от типа сиденья.





    Обозначения на рисунке 
    *A
    		для типа A
    		*B
    		для типа B
    *1
    		Щипцы для витковых пружин
    		-
    		-

  1. Введите в зацепление 4 захвата, чтобы установить 2 держателя подголовника.




  1. Для моделей с боковой подушкой безопасности переднего сиденья:

    1. Подсоедините держатель обивки спинки сиденья к подушке спинки сиденья.

    2. Установите держатель обивки спинки сиденья на раму сиденья и закрепите гайкой.

      Момент затяжки:
      5,5 N*m{ 56 kgf*cm , 49 in.*lbf }

      ПРИМЕЧАНИЕ:
      После сборки обивки сиденья убедитесь в том, что лента боковой подушки безопасности не перекручена.



  1. Закройте 2 крепления, а затем закройте обивку спинки сиденья.




    Обозначения на рисунке 
    *A
    		для сиденья с тканевой обивкой
    		*B
    		для сиденья с кожаной обивкой




  1. С помощью щипцов для витковых пружин установите 3 новые витковые пружины.
    Обозначения на рисунке 
    *1
    		Щипцы для витковых пружин

  1. Подсоедините разъем подогревателя сиденья и закрепите зажим жгута проводов.

  1. Установите резиновую ленту на пружину подушки сиденья.
