1. СЛЕЙТЕ ОХЛАЖДАЮЩУЮ ЖИДКОСТЬ ДВИГАТЕЛЯ

  1. Для моделей с 1GR-FE:
    Слейте охлаждающую жидкость двигателя (Нажмите здесь).

  1. Для моделей с 2TR-FE:
    Слейте охлаждающую жидкость двигателя (Нажмите здесь).

  1. Для моделей с 1KD-FTV с DPF:
    Слейте охлаждающую жидкость двигателя (Нажмите здесь).

  1. Для моделей с 1KD-FTV без DPF:
    Слейте охлаждающую жидкость двигателя (Нажмите здесь).

  1. Для моделей с 5L-E:
    Слейте охлаждающую жидкость двигателя (Нажмите здесь).



2. СНИМИТЕ РАДИАТОР В СБОРЕ

  1. Для моделей с двигателем 1GR-FE:
    Снимите радиатор в сборе (Нажмите здесь).

  1. Для моделей с 2TR-FE:
    Снимите радиатор в сборе (Нажмите здесь).

  1. Для моделей с 1KD-FTV с DPF:
    Снимите радиатор в сборе (Нажмите здесь).

  1. Для моделей с 1KD-FTV без DPF:
    Снимите радиатор в сборе (Нажмите здесь).

  1. Для моделей с 5L-E:
    Снимите радиатор в сборе (Нажмите здесь).



3. СОБЕРИТЕ ХЛАДАГЕНТ ИЗ СИСТЕМЫ ОХЛАЖДЕНИЯ

  1. Запустите двигатель.

  1. Включите компрессор системы кондиционирования в следующих условиях:
    Параметр / Устройство
    		Состояние
    Частота вращения коленчатого вала двигателя
    		Холостой ход
    Время работы
    		3 мин или более
    Состояние выключателя системы кондиционирования
    		Вкл
    Состояние переключателя вентилятора
    		HI
    Заданная температура
    		MAX COOL
    В результате большая часть компрессорного масла из разных узлов системы кондиционирования будет собрана в компрессоре системы кондиционирования.

    ПРИМЕЧАНИЕ:
    Нет необходимости включать компрессор системы кондиционирования, если кондиционер не работает вследствие блокировки компрессора и иных причин.

  1. Выключите двигатель.

  1. Соберите хладагент из системы кондиционирования с помощью установки регенерации хладагента.

    УКАЗАНИЕ:
    Устройство сбора/регенерации хладагента следует использовать в соответствии с руководством по эксплуатации изготовителя.



4. ОТСОЕДИНИТЕ ТРУБОПРОВОД ВЫСОКОГО ДАВЛЕНИЯ В СБОРЕ



  1. Выверните болт и отсоедините трубопровод высокого давления от конденсатора системы кондиционирования.
    Обозначения на рисунке 
    *A
    		для моделей с 1GR-FE
    *B
    		для моделей с 2TR-FE
    *C
    		для моделей с двигателями 1KD-FTV, 5L-E

    ПРИМЕЧАНИЕ:
    1. Выворачивая болт, следите за тем, чтобы ни один из инструментов не касался трубопровода.
    2. Выворачивая болт, удерживайте часть трубопровода рядом с разъемом.

  1. Снимите с трубопровода высокого давления кольцевое уплотнение.

    ПРИМЕЧАНИЕ:
    Для предотвращения проникновения влаги и загрязняющих веществ герметизируйте отверстия отсоединенных деталей с помощью виниловой ленты.



5. ОТСОЕДИНИТЕ ТРУБКУ СИСТЕМЫ КОНДИЦИОНИРОВАНИЯ В СБОРЕ



  1. Выверните болт и отсоедините трубку системы кондиционирования в сборе от конденсатора системы кондиционирования.

    ПРИМЕЧАНИЕ:
    1. Выворачивая болт, следите за тем, чтобы ни один из инструментов не касался трубопровода.
    2. Выворачивая болт, удерживайте часть трубопровода рядом с разъемом.

  1. Снимите кольцевое уплотнение с трубки системы кондиционирования в сборе.

    ПРИМЕЧАНИЕ:
    Для предотвращения проникновения влаги и загрязняющих веществ герметизируйте отверстия отсоединенных деталей с помощью виниловой ленты.



6. ОТСОЕДИНИТЕ КРОНШТЕЙН КОНДЕНСАТОРА СИСТЕМЫ КОНДИЦИОНИРОВАНИЯ № 1



  1. Выверните болт и отсоедините кронштейн конденсатора системы кондиционирования № 1.



7. ОТСОЕДИНИТЕ КРОНШТЕЙН КОНДЕНСАТОРА СИСТЕМЫ КОНДИЦИОНИРОВАНИЯ № 2



  1. Выверните болт и отсоедините кронштейн конденсатора системы кондиционирования № 2.



8. СНИМИТЕ КОНДЕНСАТОР СИСТЕМЫ КОНДИЦИОНИРОВАНИЯ В СБОРЕ



  1. Выверните 3 болта и снимите конденсатор системы кондиционирования, как показано на рисунке.



9. СНИМИТЕ ОСУШИТЕЛЬ СИСТЕМЫ КОНДИЦИОНИРОВАНИЯ



  1. С помощью торцевой шестигранной головки на 14 мм снимите крышку с модулятора.




  1. Снимите с крышки кольцевое уплотнение.




  1. С помощью плоскогубцев извлеките осушитель системы кондиционирования.
