DTC    B2325    Неисправность на шине передачи данных LIN

для подготовки Нажмите здесь


ОПИСАНИЕ

Главный ЭБУ кузова (бортовой ЭБУ сети мультиплексной связи) периодически проверяет шину передачи данных LIN между устройствами, связанными с дверями, реле управления двойной блокировкой дверей*1 и приводным механизмом люка*2. Код DTC B2325 регистрируется, когда 3 раза подряд обнаруживается нарушение обмена данными по шине LIN между устройствами, связанными с дверями, реле управления двойной блокировкой дверей*1 и приводным механизмом люка*2.
  1. *1: Для моделей с системой двойной блокировки
  2. *2: Для моделей с люком

  1. Главный переключатель сети мультиплексной связи
  2. Электродвигатель стеклоподъемника левой передней двери в сборе
  3. Электродвигатель стеклоподъемника правой передней двери в сборе
  4. Электродвигатель стеклоподъемника левой задней двери в сборе*1
  5. Электродвигатель стеклоподъемника правой задней двери в сборе*1

  9. Жгут или разъем
  1. *1: Для 5-дверных моделей
  2. *2: Для моделей с системой двойной блокировки
  3. *3: Для моделей с люком

СХЕМА СОЕДИНЕНИЙ




ПОСЛЕДОВАТЕЛЬНОСТЬ ПРОВЕРКИ

ПРИМЕЧАНИЕ:
При использовании портативного диагностического прибора для диагностики с выключенным зажиганием:
Подключите портативный диагностический прибор к автомобилю и с интервалом в 1,5 секунды включайте и выключайте освещение дверного проема до тех пор, пока не установится связь между диагностическим прибором и автомобилем.

УКАЗАНИЕ:
Если код DTC B2325 и код DTC прекращения обмена данными по шине LIN выводятся одновременно, в первую очередь выполните поиск неисправностей для кода DTC прекращения обмена данными по шине LIN. Затем выполните диагностику для кода DTC B2325.
