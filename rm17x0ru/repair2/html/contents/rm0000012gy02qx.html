НАСТРОЙКА ПАРАМЕТРОВ С ПОМОЩЬЮ ПОРТАТИВНОГО ДИАГНОСТИЧЕСКОГО ПРИБОРА (ДЛЯ СПРАВКИ)

УКАЗАНИЕ:
Можно настроить следующие параметры.

ПРИМЕЧАНИЕ:
  1. Если клиент требует изменить настройки, прежде всего следует убедиться, что значения соответствующих параметров могут изменяться.
  2. Прежде чем приступать к настройке параметров, запишите текущие установки.

  1. Подсоедините портативный диагностический прибор к разъему DLC3.

  2. Включите зажигание.

  3. Включите портативный диагностический прибор.

  4. Войдите в следующие меню: Customize Setting / Power Window.

  5. Проверьте состояние в соответствии с таблицей ниже.

    Power Window (Электрические стеклоподъемники) 
    Информация на дисплее прибора
    		Значение по умолчанию
    		Описание
    		Настройка
    Door Key Linked P/W Up
    		OFF (ВЫКЛ)
    		Данная функция используется для закрывания окон при помощи механического ключа.
    		ON (ВКЛ) или OFF (ВЫКЛ)
    Door Key Linked P/W Down
    		OFF (ВЫКЛ)
    		Данная функция используется для открывания окон при помощи механического ключа.
    		ON (ВКЛ) или OFF (ВЫКЛ)
    P/W Up w/Transmitter
    		OFF (ВЫКЛ)
    		Данная функция используется для закрывания окон при помощи дистанционного передатчика.
    		ON (ВКЛ) или OFF (ВЫКЛ)
    P/W Down w/ Transmitter
    		OFF (ВЫКЛ)
    		Данная функция используется для открывания окон при помощи дистанционного передатчика.
    		ON (ВКЛ) или OFF (ВЫКЛ)
