1. INSTALL INPUT SHAFT FRONT BEARING



  1. Using SST and a press, install a new front bearing.

    SST
    09950-60010   (09951-00400)
    09950-70010   (09951-07100)




  1. Select a snap ring that will allow minimal axial play.

    Standard clearance:
    0.10 mm (0.0039 in.) or less

    Standard Front Bearing Shaft Snap Ring:
    Mark
    		Thickness
    0
    		2.05 to 2.10 mm (0.0807 to 0.0827 in.)
    1
    		2.10 to 2.15 mm (0.0827 to 0.0847 in.)
    2
    		2.15 to 2.20 mm (0.0847 to 0.0866 in.)
    3
    		2.20 to 2.25 mm (0.0866 to 0.0886 in.)
    4
    		2.25 to 2.30 mm (0.0886 to 0.0906 in.)
    5
    		2.30 to 2.35 mm (0.0906 to 0.0925 in.)




  1. Using a snap ring expander, install the snap ring.



2. INSTALL INPUT SHAFT BEARING



  1. Coat the 13 input shaft bearings with MP grease and install them to the input shaft.



3. INSTALL NO. 2 SYNCHRONIZER RING



  1. Coat the No. 2 synchronizer ring with gear oil and install it to the input shaft.
