ОПИСАНИЕ

Активная управляемая подвеска двигателя (ACM) снижает вибрацию двигателя при работе на низкой частоте вращения коленчатого вала двигателя с помощью VSV для опоры двигателя. VSV управляется импульсным сигналом, передаваемым из ECM. Для эффективного снижения вибрации двигателя частота данных импульсных сигналов согласовывается с частотой вращения коленчатого вала двигателя.

СХЕМА СОЕДИНЕНИЙ




ПОСЛЕДОВАТЕЛЬНОСТЬ ПРОВЕРКИ

ПРИМЕЧАНИЕ:
  1. Перед выполнением следующей процедуры проверки проверьте предохранители цепей, относящихся к данной системе.
  2. После замены новый ЕСМ необходимо зарегистрировать (Нажмите здесь) и инициализировать (Нажмите здесь).
  3. После замены нагнетающего топливного насоса в сборе необходимо выполнить инициализацию ЕСМ (Нажмите здесь).
  4. После замены форсунки необходимо выполнить регистрацию ЕСМ (Нажмите здесь).

1.ВЫПОЛНИТЕ ДИАГНОСТИКУ В РЕЖИМЕ ACTIVE TEST С ПОМОЩЬЮ ПОРТАТИВНОГО ДИАГНОСТИЧЕСКОГО ПРИБОРА

  1. Прогрейте двигатель.

  1. Подсоедините портативный диагностический прибор к DLC3.

  1. Установите рычаг переключения передач в положение N или Р.

  1. Войдите в следующие меню: Powertrain / Engine and ECT / Active Test / Control the ACM Inhibit.

  1. Удерживая верхнюю часть рулевого колеса, выполните испытание Active Test, установив для него вместо значения OFF (ВЫКЛ) значение ON (ВКЛ). Проверьте вибрацию двигателя.
