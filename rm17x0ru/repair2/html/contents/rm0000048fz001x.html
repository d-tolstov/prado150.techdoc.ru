1. INSTALL OUTPUT SHAFT ASSEMBLY



  1. Install the output shaft to the intermediate plate by pushing on the output shaft and tapping on the intermediate plate with a plastic-faced hammer.

  1. Using a snap ring expander, install the snap ring to the center bearing.

    УКАЗАНИЕ:
    Make sure the shaft snap ring is flush with the intermediate plate surface.



2. INSTALL INPUT SHAFT ASSEMBLY



  1. Coat the input shaft and No. 2 synchronizer ring with gear oil, and install them to the output shaft.



3. INSTALL COUNTER GEAR ASSEMBLY



  1. Coat the input shaft and No. 2 synchronizer ring with gear oil.

  1. Temporarily install the counter gear, input shaft and a new center bearing to the intermediate plate.

    ПРИМЕЧАНИЕ:
    Install the bearing (with snap ring) to the rear side of the counter gear.




  1. Using a plastic-faced hammer, install the center bearing to the intermediate plate by tapping the outer race to the counter gear center bearing.

    УКАЗАНИЕ:
    Hold the front side of the counter gear by hand.

  1. Using a snap ring expander, install the counter shaft center bearing snap ring to the counter shaft center bearing.



4. INSTALL OUTPUT SHAFT REAR BEARING RETAINER

  1. Using a T40 "TORX" socket, install the output shaft rear bearing retainer to the intermediate plate with the 4 screws.

    Момент затяжки:
    18 N*m{ 184 kgf*cm , 13 ft.*lbf }



5. INSTALL REVERSE IDLER GEAR SUB-ASSEMBLY

  1. Install the reverse idler gear shaft and reverse idler gear to the intermediate plate.

  1. Install the reverse idler gear shaft stopper to the intermediate plate with the bolt.

    Момент затяжки:
    17 N*m{ 173 kgf*cm , 13 ft.*lbf }



6. INSTALL REVERSE SHIFT ARM BRACKET

  1. Install the reverse shift arm bracket to the intermediate plate with the 2 bolts.

    Момент затяжки:
    18 N*m{ 184 kgf*cm , 13 ft.*lbf }



7. INSTALL 5TH GEAR BEARING INNER RACE LOCK BALL

  1. Install the lock ball.



8. INSTALL 5TH GEAR THRUST WASHER

  1. Install the 5th gear thrust washer.



9. INSTALL NO. 3 TRANSMISSION HUB SLEEVE

  1. Coat the counter shaft 5th gear with gear oil.





  1. Install the counter 5th gear, counter shaft 5th gear bearing, 3 synchromesh shifting keys, 2 synchromesh shifting key springs and No. 3 transmission hub sleeve.

10. INSTALL COUNTER SHAFT 5TH GEAR



  1. Install the counter 5th gear bearing to the 5th gear.

  1. Install the counter shaft 5th gear to the counter gear.




  1. Temporarily install the No. 1 synchronizer ring on the No. 5 gear spline piece.




  1. Remove the intermediate plate from the vise.

  1. Stand the transmission as shown in the illustration.




  1. Using a press and 22 mm socket wrench, install the No. 5 gear spline piece with the No. 1 synchronizer ring slots aligned with the shifting keys.

  1. Fix the intermediate plate in a vise between aluminum plates.




  1. Select a snap ring that will allow minimal axial play.

    Standard clearance:
    0.10 mm (0.00394 in.) or less

    Standard Counter Gear Rear Shaft Snap Ring:
    Mark
    		Thickness
    A
    		2.80 to 2.85 mm (0.110 to 0.112 in.)
    B
    		2.85 to 2.90 mm (0.112 to 0.114 in.)
    C
    		2.90 to 2.95 mm (0.114 to 0.116 in.)
    D
    		2.95 to 3.00 mm (0.116 to 0.118 in.)
    E
    		3.00 to 3.05 mm (0.118 to 0.120 in.)
    F
    		3.05 to 3.10 mm (0.120 to 0.122 in.)
    G
    		3.10 to 3.15 mm (0.122 to 0.124 in.)




  1. Using a brass bar and hammer, tap the snap ring to the counter gear.



11. INSPECT COUNTER SHAFT 5TH GEAR THRUST CLEARANCE



  1. Using a feeler gauge, measure the counter shaft 5th gear thrust clearance.

    Standard clearance:
    0.10 to 0.30 mm (0.00394 to 0.0118 in.)

    1. If the result is not as specified, replace the counter shaft 5th gear.



12. INSTALL NO. 2 GEAR SHIFT FORK SHAFT



  1. Install the No. 2 gear shift fork and No. 1 gear shift fork.

  1. Pass the No. 2 gear shift fork shaft through the intermediate plate, No. 2 gear shift fork and No. 1 gear shift fork to install it.




  1. Using a brass bar and hammer, tap the shaft snap ring onto the No. 2 gear shift fork shaft.

  1. Install the bolt to the No. 2 gear shift fork.

    Момент затяжки:
    20 N*m{ 204 kgf*cm , 15 ft.*lbf }



13. INSTALL NO. 2 SHIFT INTERLOCK PIN



  1. Coat the No. 2 shift interlock pin with MP grease and install it to the No. 1 gear shift fork shaft.



14. INSTALL NO. 1 GEAR SHIFT FORK SHAFT

  1. Coat the No. 1 shift interlock pin with MP grease.




  1. Using a magnet hand, install the No. 1 shift interlock pin to the intermediate plate.

  1. Pass the No. 1 gear shift fork shaft through the intermediate plate and No. 2 gear shift fork to install it.

  1. Install the bolt to the No. 1 shift fork.

    Момент затяжки:
    20 N*m{ 204 kgf*cm , 15 ft.*lbf }




  1. Using pliers, install the shaft snap ring to the No. 1 gear shift fork shaft.



15. INSTALL REVERSE SHIFT ARM



  1. Install the shift arm shoe E-ring and the shift arm shoe to the reverse shift arm.




  1. Install the reverse shift arm E-ring and reverse shift fork to the reverse shift arm.

  1. Install the reverse shift arm to the reverse shift arm bracket.



16. INSTALL NO. 2 SHIFT INTERLOCK PIN



  1. Coat the No. 2 shift interlock pin with MP grease, and install it to the No. 3 gear shift fork shaft.



17. INSTALL NO. 3 SHIFT INTERLOCK PIN



  1. Coat the No. 3 shift interlock pin with MP grease.

  1. Using a magnet hand, install the No. 3 shift interlock pin to the intermediate plate.



18. INSTALL NO. 3 GEAR SHIFT FORK SHAFT

  1. Install the No. 3 gear shift fork shaft through the reverse shift fork and intermediate plate.




  1. Install the reverse shift head ring to the reverse shift fork shaft.

  1. Using a 5 mm pin punch and a hammer, tap in the slotted spring pin to the No. 3 shift fork shaft.



19. INSTALL NO. 5 GEAR SHIFT FORK SHAFT



  1. Install the No. 5 gear shift fork shaft and reverse shift head.

  1. Using a 5 mm pin punch and hammer, tap in the slotted spring pin to the No. 3 shift fork shaft.



20. INSTALL NO. 4 GEAR SHIFT FORK SHAFT



  1. Coat the 2 shift detent balls with MP grease.

  1. Using a magnet hand, install the 2 shift detent balls to the intermediate plate.




  1. Pass the No. 4 gear shift fork shaft through the reverse shift fork, intermediate plate and reverse shift head to install it.

  1. Install a new No. 3 gear shift fork with the bolt.

    Момент затяжки:
    20 N*m{ 204 kgf*cm , 15 ft.*lbf }



21. INSTALL NO. 2 SHIFT DETENT BALL



  1. Coat the No. 2 detent ball with MP grease.

  1. Install the No. 2 detent ball and compression spring to the intermediate plate.

  1. Coat the spring seat with adhesive.

    Adhesive:
    Toyota Genuine Adhesive 1344, Three Bond 1344 or equivalent

  1. Using a T40 "TORX" socket, install the spring seat to the intermediate plate.

    Момент затяжки:
    19 N*m{ 194 kgf*cm , 14 ft.*lbf }



22. INSTALL SHIFT DETENT BALL

  1. Coat the 3 shift detent balls with MP grease, and then install them and the 3 compression springs to the intermediate plate.

  1. Coat the spring seat and 2 ball plugs with adhesive.

    Adhesive:
    Toyota Genuine Adhesive 1344, Three Bond 1344 or equivalent

  1. Using a T40 "TORX" socket, install the spring seat and 2 ball plugs to the intermediate plate.

    Момент затяжки:
    19 N*m{ 194 kgf*cm , 14 ft.*lbf }

  1. Using a snap ring expander, install the snap ring to the counter gear front bearing.



25. INSTALL FRONT BEARING SHAFT SNAP RING

  1. Using a snap ring expander, install the snap ring to the input shaft front bearing.



26. INSTALL TRANSMISSION FRONT BEARING RETAINER OIL SEAL



  1. Using SST, press in a new oil seal to the front bearing retainer.

    SST
    09950-60010   (09951-00440, 09952-06010)

    Standard depth:
    12.2 to 13.2 m (0.480 to 0.519 in.)

    Text in Illustration 
    *a

    ПРИМЕЧАНИЕ:
    Do not damage the oil seal lip.

  1. Apply a light coat of MP grease to the lip of the oil seal.



27. INSTALL FRONT BEARING RETAINER

  1. Install a new gasket and the front bearing retainer to the transmission case.

  1. Install the 8 bolts.

    Момент затяжки:
    17 N*m{ 173 kgf*cm , 13 ft.*lbf }



28. INSTALL TRANSFER ADAPTER OIL SEAL

    ПРИМЕЧАНИЕ:

  1. Apply a light coat of MP grease to the lip of the oil seal.

  1. Using a 5 mm pin punch and hammer, tap in the slotted spring pin to the transfer adaptor.

  1. Using a T40 "TORX" socket, install the plug to the transfer adaptor.
