1. INSTALL TRANSFER ASSEMBLY

  1. Для моделей с 1KD-FTV:
    Закрепите раздаточную коробку в сборе с помощью 8 болтов и 2 кронштейнов.

    Момент затяжки:
    24 N*m{ 245 kgf*cm , 18 ft.*lbf }

    ПРИМЕЧАНИЕ:
    Соблюдайте осторожность, чтобы не повредить задний сальник переходника шлицами первичной шестерни раздаточной коробки.

  1. Кроме моделей с 1KD-FTV:
    Закрепите раздаточную коробку в сборе с помощью 8 болтов и кронштейна.

    Момент затяжки:
    24 N*m{ 245 kgf*cm , 18 ft.*lbf }

    ПРИМЕЧАНИЕ:
    Соблюдайте осторожность, чтобы не повредить задний сальник переходника шлицами первичной шестерни раздаточной коробки.



2. INSTALL MANUAL TRANSMISSION UNIT ASSEMBLY

  1. Align the input shaft with the clutch disc and install the transmission to the engine.

  1. Install the 4 bolts.

    Момент затяжки:
    72 N*m{ 730 kgf*cm , 52 ft.*lbf }



3. INSTALL STIFFENER PLATE LH



  1. Install the stiffener plate with the 4 bolts.

    Момент затяжки:
    for bolt A:
    37 N*m{ 380 kgf*cm , 27 ft.*lbf }
    for bolt B:
    69 N*m{ 701 kgf*cm , 50 ft.*lbf }



4. INSTALL STIFFENER PLATE RH



  1. Install the stiffener plate with the 4 bolts.

    Момент затяжки:
    for bolt A:
    37 N*m{ 380 kgf*cm , 27 ft.*lbf }
    for bolt B:
    69 N*m{ 701 kgf*cm , 50 ft.*lbf }



5. INSTALL REAR NO. 1 ENGINE MOUNTING INSULATOR

  1. Install the mounting insulator with the 4 bolts.

    Момент затяжки:
    65 N*m{ 663 kgf*cm , 48 ft.*lbf }



6. INSTALL NO. 3 FRAME CROSSMEMBER SUB-ASSEMBLY

  1. Install the frame crossmember with the 4 bolts and 4 nuts.

    Момент затяжки:
    72 N*m{ 734 kgf*cm , 53 ft.*lbf }

  1. Install the 4 bolts to the rear No. 1 engine mounting insulator.

    Момент затяжки:
    30 N*m{ 306 kgf*cm , 22 ft.*lbf }



7. INSTALL FRONT SUSPENSION MEMBER BRACKET LH

  1. Install the member bracket with the 4 bolts.

    Момент затяжки:
    33 N*m{ 337 kgf*cm , 24 ft.*lbf }



8. INSTALL FRONT SUSPENSION MEMBER BRACKET RH

  1. Install the member bracket with the 4 bolts.

    Момент затяжки:
    33 N*m{ 337 kgf*cm , 24 ft.*lbf }



9. INSTALL STARTER ASSEMBLY

  1. Install the starter (See page Нажмите здесь).



10. CONNECT CLUTCH RELEASE CYLINDER ASSEMBLY

  1. Connect the release cylinder with the 2 bolts.

    Момент затяжки:
    12 N*m{ 120 kgf*cm , 9 ft.*lbf }



11. INSTALL TRANSFER AND MANUAL TRANSMISSION BREATHER HOSE SUB-ASSEMBLY

  1. Install the 3 breather hoses to the shift lever retainer and transfer adapter and attach the clamp.



12. CONNECT WIRE HARNESS

  1. Connect the 2 connectors and attach the 4 clamps.



13. INSTALL FRONT EXHAUST PIPE ASSEMBLY

  1. Install the front exhaust pipe (See page Нажмите здесь).



14. INSTALL PROPELLER SHAFT ASSEMBLY

  1. Install the propeller shaft (See page Нажмите здесь).



15. INSTALL FRONT PROPELLER SHAFT ASSEMBLY

  1. Install the front propeller shaft (See page Нажмите здесь).



16. ADD MANUAL TRANSMISSION OIL


17. INSTALL TRANSFER CASE LOWER PROTECTOR

  1. Install the transfer case lower protector with the 4 bolts.

    Момент затяжки:
    18 N*m{ 184 kgf*cm , 13 ft.*lbf }



18. INSTALL FLOOR SHIFT SHIFT LEVER ASSEMBLY

  1. Cover the shift lever cap with a cloth.

  1. Press down on the shift lever cap and rotate it clockwise to install it.

    УКАЗАНИЕ:
    Apply MP grease to the tip of the shift lever.



19. INSTALL NO. 1 SHIFT AND SELECT LEVER BOOT

  1. Install the boot with the 4 screws and 2 clips.



20. INSTALL CONSOLE PANEL SUB-ASSEMBLY

  1. Подсоедините все разъемы.

  1. Введите в зацепление 8 фиксаторов и 2 захвата и установите облицовку панели пола.



21. INSTALL SHIFT LEVER KNOB SUB-ASSEMBLY
