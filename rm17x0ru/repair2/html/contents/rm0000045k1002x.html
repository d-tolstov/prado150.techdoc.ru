1. INSTALL FUEL SENDER GAUGE ASSEMBLY



  1. Set the fuel sender gauge on the fuel sub-tank. Then slide the fuel sender gauge downward to install it.
    Text in Illustration 
    *1
    		Fuel Sub-Tank

    		Slide

  1. Connect the fuel sender gauge connector.



2. INSTALL FUEL SENDER GAUGE ASSEMBLY (for Fuel Sub Tank)



  1. Set the fuel sender gauge on the fuel gauge support. Then slide the fuel sender gauge upward to install it.
    Text in Illustration 
    *1
    		Fuel Gauge Support

    		Slide

    ПРИМЕЧАНИЕ:
    Make sure the part labeled A in the illustration is securely inserted into the hole of the fuel gauge support before sliding the fuel sender gauge upward.




  1. Install the 2 lead wires with the 2 screws.

    Момент затяжки:
    2.0 N*m{ 20 kgf*cm , 18 in.*lbf }

    Text in Illustration 
    *1
    		Lead Wire (Black)
    *2
    		Lead Wire (Red)

    ПРИМЕЧАНИЕ:
    Be sure to install the lead wires in the correct locations according to their color as shown in the illustration.

  1. for Double Tank Type:

    2. for Fuel Sub Tank:
