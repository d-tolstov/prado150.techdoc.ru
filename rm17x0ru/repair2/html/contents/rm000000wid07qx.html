РАБОТА В АВАРИЙНОМ РЕЖИМЕ

  1. При сбое передачи данных по любым шинам CAN (шинам передачи данных) включается функция работы в аварийном режиме. Функция работы в аварийном режиме, определенная для каждой системы, работает таким образом, чтобы предотвратить выход этой системы из строя.

  2. Влияет на работу различных систем, когда обмен данных невозможен (дополнительная информация приведена на страницах, соответствующих отдельным системам).

    		Уведомление пользователя
    		Диагностический код неисправности DTC
