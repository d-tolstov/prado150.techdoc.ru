1. INSTALL STARTER ASSEMBLY

  1. Install the starter with the 2 bolts and nut.

    Момент затяжки:
    68 N*m{ 693 kgf*cm , 50 ft.*lbf }

  1. Connect the starter wire with the nut.

    Момент затяжки:
    9.8 N*m{ 100 kgf*cm , 87 in.*lbf }

  1. Install the terminal cap.

  1. Connect the starter connector.



2. CONNECT CLUTCH FLEXIBLE HOSE BRACKET

  1. Connect the clutch flexible hose bracket with the bolt.

    Момент затяжки:
    72 N*m{ 730 kgf*cm , 53 ft.*lbf }



3. CONNECT CLUTCH RELEASE CYLINDER ASSEMBLY

  1. Connect the release cylinder with the 2 bolts.

    Момент затяжки:
    12 N*m{ 120 kgf*cm , 9 ft.*lbf }



4. INSTALL REAR ENGINE UNDER COVER ASSEMBLY

  1. Install the rear engine under cover with the 4 bolts.

    Момент затяжки:
    29 N*m{ 296 kgf*cm , 21 ft.*lbf }



5. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

ПРИМЕЧАНИЕ:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (See page Нажмите здесь).
