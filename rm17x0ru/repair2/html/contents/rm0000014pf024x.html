1. INSTALL GENERATOR ASSEMBLY

  1. Temporarily install the generator with the 2 bolts.

  1. Connect the vacuum pump oil outlet hose.

  1. Install 2 new gaskets and the vacuum pump oil inlet hose with the union bolt.

    Момент затяжки:
    14 N*m{ 140 kgf*cm , 10 ft.*lbf }

  1. Connect the vacuum pump hose.

  1. Attach the vacuum pump oil inlet hose to the cord clip.

  1. Install the generator wire with the nut.

    Момент затяжки:
    9.8 N*m{ 100 kgf*cm , 87 in.*lbf }

  1. Install the terminal cap.

  1. Connect the generator connector.



2. CONNECT COOLER COMPRESSOR ASSEMBLY



  1. Temporarily install the cooler compressor with the 2 bolts.

  1. Temporarily install the idle pulley bracket with the 4 bolts.

  1. Tighten the 6 bolts in the sequence shown in the illustration.

    Момент затяжки:
    for bolt A:
    45 N*m{ 459 kgf*cm , 33 ft.*lbf }
    except bolt A:
    25 N*m{ 250 kgf*cm , 18 ft.*lbf }

  1. Connect the wire harness with the bolt.

    Момент затяжки:
    13 N*m{ 131 kgf*cm , 9 ft.*lbf }

  1. Connect the cooler compressor connector.



3. INSTALL FAN AND GENERATOR V BELT

  1. Install the fan and generator V belt (See page Нажмите здесь).



4. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL
