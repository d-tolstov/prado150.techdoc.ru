УКАЗАНИЕ:
When replacing the filter, use TOYOTA genuine parts or equivalent parts.



1. REMOVE FUEL FILTER ASSEMBLY



  1. Disconnect the 2 fuel hoses.

  1. Disconnect the level warning switch connector.

  1. Remove the 2 nuts and fuel filter.



2. DRAIN FUEL



  1. Loosen the fuel filter drain plug and drain the fuel from the fuel filter.



3. REMOVE FUEL FILTER CASE



  1. Detach the level warning switch connector clamp from the fuel filter cap.

  1. Remove the 2 bolts, fuel filter case and fuel filter gasket.



4. REMOVE LEVEL WARNING SWITCH



  1. Clamp the fuel filter cap in a vise between aluminum plates.

    ПРИМЕЧАНИЕ:
    Be careful not to damage the fuel filter cap.

  1. Using pliers, remove the level warning switch.

    ПРИМЕЧАНИЕ:
    Be careful not to damage the level warning switch.

  1. Remove the warning switch gasket from the level warning switch.



5. REMOVE FUEL FILTER ELEMENT ASSEMBLY



  1. Using SST, remove the fuel filter element.

    SST
    09228-64010  



6. INSTALL FUEL FILTER ELEMENT ASSEMBLY

  1. Check and clean the installation surface of the fuel filter element.

  1. Apply fuel to the gasket of a new fuel filter element.

  1. Lightly screw the fuel filter element into place, and tighten it until the gasket comes into contact with the seat.




  1. Tighten the fuel filter element an additional 3/4 turn by hand.



7. INSTALL LEVEL WARNING SWITCH

  1. Install a new level warning switch gasket to the level warning switch.

  1. Apply fuel to the level warning switch gasket of the level warning switch.




  1. Install the level warning switch to the fuel filter element and tighten it by hand.
    Text in Illustration 
    *a
    		Front
    *b
    		Within 28.4°

    УКАЗАНИЕ:
    Install the level warning switch within the angle range shown in the illustration.



8. INSTALL FUEL FILTER CASE

  1. Install the fuel filter gasket and fuel filter case to the fuel filter element.




  1. Tighten the 2 bolts.

    Момент затяжки:
    3.4 N*m{ 35 kgf*cm , 30 in.*lbf }

    УКАЗАНИЕ:
    Install the level warning switch within the angle range shown in the illustration.

  1. Attach the level warning switch connector clamp to the fuel filter cap.
    Text in Illustration 
    *a
    		Front
    *b
    		0 to 20°



9. INSTALL FUEL FILTER ASSEMBLY

  1. Install the fuel filter with the 2 nuts.

    Момент затяжки:
    18 N*m{ 184 kgf*cm , 13 ft.*lbf }




  1. Connect the level warning switch connector.

  1. Connect the 2 fuel hoses.
    Text in Illustration 
    *a
    		Upper



10. BLEED AIR FROM FUEL SYSTEM



  1. Using the hand pump mounted on the fuel filter cap, bleed the air from the fuel system. Continue pumping until the pump resistance increases.



11. INSPECT FOR FUEL LEAK

  1. Check that there are no fuel leaks anywhere in the fuel system after performing maintenance.

    УКАЗАНИЕ:
    When checking for fuel leaks, make sure that there is pressure in the fuel line.
