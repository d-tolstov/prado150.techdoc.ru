BUS CHECK (COMMUNICATION MALFUNCTION DTC) (проверка шин (коды DTC сбоев связи))
На экране портативного диагностического прибора могут выводиться только коды DTC мультиплексной шины CAN для каждого ЭБУ.



  1. Выберите в окне портативного диагностического прибора "System Select" (выбор системы) режим "Bus Check" (проверка шин).




  2. В окне "Bus Check" (проверка шин) выберите пункт "Communication Malfunction DTC" (коды DTC сбоев связи), а затем нажмите "Enter" (ввод).




  3. Выберите систему, для которой необходимо выполнить проверку DTC, и нажмите "Details" (детали).




  4. Выводятся коды DTC, относящиеся к мультиплексной шине CAN.


BUS CHECK (COMMUNICATION BUS CHECK) (проверка шин (проверка шин связи))

УКАЗАНИЕ:
На экране портативного диагностического прибора отображаются блоки ЭБУ и датчики, должным образом подключенные к системе передачи данных CAN.




  1. В окне "Bus Check" (проверка шин) выберите пункт "Communication Bus Check" (проверка шин связи).

  2. На экране отображаются блоки ЭБУ и датчики, должным образом подключенные к системе передачи данных CAN.

    УКАЗАНИЕ:
    1. Если должным образом подключенные ЭБУ или датчики не отображаются, система перешла в режим прекращения обмена данными (Нажмите здесь).


    2. В качестве значения по умолчанию в комбинированном списке отображается значение "ALL". При проверке ЭБУ (датчиков), подключенных к каждой шине, раскройте комбинированный список и выберите шину, подлежащую проверке, из раскрывающегося списка. Раскрывающийся список содержит значения "ALL", "V Bus" и наименования остальных шин.

  2. При выводе кодов DTC для системы передачи данных CAN неисправность невозможно установить только посредством кодов DTC. Выполните поиск неисправности в порядке, описанном в разделе "Процедура поиска неисправностей".

ТАБЛИЦА КОМБИНАЦИЙ КОДОВ DTC

УКАЗАНИЕ:
  1. ○: Сохранен.
  2. X: При обрыве в одной из сторон вспомогательной шины может сохраняться данный код.
  3. -: Не сохраняется.
  4. *1: Сохраняется только в случае восстановления обмена данными. Не может быть сохранен при прекращении обмена данными.
  5. *2: Выводится только при наличии неисправности и при включенном зажигании.
  6. *3: за исключением моделей с 5L-E
  7. *4: для моделей с системой курсовой устойчивости (с вакуумным усилителем тормозной системы)
  8. *5: для моделей без системы курсовой устойчивости (с вакуумным усилителем тормозной системы)
