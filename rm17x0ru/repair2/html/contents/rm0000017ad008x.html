1. INSTALL OIL FILTER UNION

  1. Using a 12 mm socket hexagon wrench, install the union.

    32 N*m{ 326 kgf*cm , 24 ft.*lbf }

  1. Install the safety valve and spring and a new gasket with the plug.

    Момент затяжки:
    36 N*m{ 367 kgf*cm , 27 ft.*lbf }
