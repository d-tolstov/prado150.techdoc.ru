1. INSTALL FUEL TANK TO FILLER PIPE HOSE



  1. Align the fuel tank side mark with the hose side mark.
    Text in Illustration 
    *1
    		Fuel Tank Side Mark
    *2
    		Hose Side Mark

  1. Install the fuel tank to filler pipe hose to the fuel tank.



2. INSTALL NO. 3 FUEL TANK PROTECTOR

  1. Install the No. 3 fuel tank protector and attach the 4 clamps.

  1. Install the 2 bolts.

    Момент затяжки:
    5.0 N*m{ 51 kgf*cm , 44 in.*lbf }



3. INSTALL FUEL TANK VENT TUBE ASSEMBLY

  1. Apply a light coat of gasoline or grease to a new gasket and install the gasket to the fuel tank.




  1. Align the protrusion of the fuel tank vent tube with the groove of the fuel tank.
    Text in Illustration 
    *1
    		Protrusion
    *2
    		Groove

    ПРИМЕЧАНИЕ:
    Be careful not to bend the arm of the fuel sender gauge.

  1. Install the fuel tank vent tube to the fuel tank.



  1. Put a new retainer on the fuel tank. While holding the fuel tank vent tube, tighten the retainer one complete turn by hand.
    Text in Illustration 
    *1
    		Start Mark (Fuel Tank Side)
    *2
    		Start Mark (Retainer Side)

    УКАЗАНИЕ:

    Make sure the start marks on the retainer and fuel tank are aligned, and then tighten the retainer.




  1. Set SST on the retainer.

    SST
    09808-14030  

    УКАЗАНИЕ:
    1. Hold the fuel tank vent tube assembly upright by hand to ensure that the gasket is not moved out of position.
    2. Engage the claws of SST securely with the retainer holes to secure SST.
    3. Install SST while pressing the claws of SST against the retainer (toward the center of SST).




  1. Using SST, tighten the retainer until the mark on the retainer is within range A on the fuel tank as shown in the illustration.

    SST
    09808-14030  
    Text in Illustration 
    *1
    		Fuel Tank Side Mark
    *2
    		Retainer Side Mark

    УКАЗАНИЕ:
    Fit the tips of SST onto the ribs of the retainer.



4. INSTALL FUEL TANK MAIN TUBE SUB-ASSEMBLY AND FUEL RETURN TUBE SUB-ASSEMBLY (for Single Tank Type)



  1. Install the fuel tank main tube and fuel return tube with the 2 fuel tube joint clips.

    ПРИМЕЧАНИЕ:


    1. Check that there are no scratches or foreign objects on the connecting parts.
    2. Check that the fuel tube joints are inserted securely.
    3. Check that the fuel tube joint clips are on the collars of the fuel tube joints.
    4. After installing the fuel tube joint clips, check that the fuel tube joints cannot be pulled off.
      Text in Illustration

      *4

      		CORRECT
      *b
      		INCORRECT

  1. Install the fuel tank main tube and fuel return tube to the fuel tank.

6. INSTALL FUEL TANK CUSHION

  1. Install 3 new fuel tank cushions to the fuel tank.



7. INSTALL FUEL TANK SUB-ASSEMBLY

  1. Set the fuel tank on a transmission jack and rise the transmission jack.

    ПРИМЕЧАНИЕ:
    Do not allow the fuel tank to contact the vehicle, especially the differential.

  1. Install the 2 fuel tank bands with the 2 pins and 2 clips.

  1. Connect the 2 fuel tank bands with the 2 bolts.

    Момент затяжки:
    40 N*m{ 408 kgf*cm , 30 ft.*lbf }

20. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

ПРИМЕЧАНИЕ:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (See page Нажмите здесь).



21. BLEED AIR FROM FUEL SYSTEM



  1. Using the hand pump mounted on the fuel filter cap, bleed the air from the fuel system. Continue pumping until the pump resistance increases.



22. INSPECT FOR FUEL LEAK

  1. Check that there are no fuel leaks anywhere in the fuel system after performing maintenance.

    УКАЗАНИЕ:
    When checking for fuel leaks, make sure that there is pressure in the fuel line.
