DATA LIST

УКАЗАНИЕ:
В режиме Data List на дисплей портативного диагностического прибора можно вывести данные о состоянии переключателей, датчиков, приводов и пр. без снятия деталей с автомобиля. Такая внешняя проверка может оказаться очень полезной, поскольку позволяет выявлять нерегулярно возникающие состояния и сигналы, не нарушая монтажа деталей и электропроводки. Считывание информации в режиме Data List на первом этапе поиска неисправностей позволяет сократить время, затрачиваемое на диагностику.

ПРИМЕЧАНИЕ:
Значения, указанные в колонке "Нормальное состояние" приведенной ниже таблицы, являются справочными. Принимая решение об исправности той или иной детали, не ориентируйтесь исключительно на них.

  1. Прогрейте двигатель.

  2. Выключите зажигание.

  3. Подсоедините портативный диагностический прибор к DLC3.

  4. Включите зажигание (IG).

  5. Включите портативный диагностический прибор.

  6. Войдите в следующие меню: Powertrain / Engine and ECT / Data List / All Data.

  7. Проверьте значения в соответствии с таблицей ниже.

ACTIVE TEST

УКАЗАНИЕ:
Выполнение активной диагностики с использованием портативного диагностического прибора дает возможность управлять реле, электровакуумными клапанами, приводами и т.д. без снятия деталей с автомобиля. Такая внешняя функциональная проверка может оказаться очень полезной, поскольку позволяет выявлять эпизодические неисправности, не нарушая монтаж деталей и электропроводки. Выполнение активной диагностики на начальном этапе поиска неисправностей позволяет сократить время, затрачиваемое на диагностику. В режиме Active Test на дисплей диагностического прибора можно вывести список параметров Data list.

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Включите зажигание (IG).

  3. Включите портативный диагностический прибор.

  4. Войдите в следующие меню: Powertrain / Engine and ECT / Active Test.

  5. В соответствии с указаниями на дисплее выполните диагностику в режиме ACTIVE TEST.

УКАЗАНИЕ:
*: Во время прокручивания коленчатого вала двигателя частота вращения измеряется в режиме Active Test для каждого цилиндра.
В данном испытании в режиме Active Test впрыск топлива во все цилиндры прекращается, и прокручивание коленчатого вала двигателя стартером выполняется в течение примерно 10 с. При этом измеряется частота вращения в каждом цилиндре. Если частота вращения в одном цилиндре выше, чем в других цилиндрах, определяется, что давление сжатия в этом цилиндре ниже, чем в других цилиндрах.

  1. Прогрейте двигатель.
  2. Выключите зажигание.
  3. Подсоедините портативный диагностический прибор к DLC3.
  4. Включите зажигание (IG) и портативный диагностический прибор.
  5. Войдите в следующие меню: Powertrain / Engine and ECT / Active Test / Check the Cylinder Compression.

    УКАЗАНИЕ:
    Если результаты отображаются ненадлежащим образом, перед выполнением испытания Active Test в режиме Data List выберите параметры для отображения. Войдите в следующие меню: Powertrain / Engine and ECT / Data List / Compression / Engine Speed of Cyl #1, Engine Speed of Cyl #2, Engine Speed of Cyl #3, Engine Speed of Cyl #4, Engine Speed of Cyl #5, Engine Speed of Cyl #6 и Av Engine Speed of All Cyl.

  6. Когда двигатель остановлен, с помощью кнопок "вправо" или "влево" установите значение ON (ВКЛ) для параметра "Check the Cylinder Compression".

    УКАЗАНИЕ:
    После выполнения описанных действий запускается испытание "Check the Cylinder Compression". Впрыск топлива для всех цилиндров запрещается, а система переходит в режим ожидания измерения частоты вращения коленчатого вала двигателя для каждого цилиндра.

  7. Прокрутите коленчатый вал двигателя стартером в течение примерно 10 с.
  8. Проверьте значение частоты вращения коленчатого вала двигателя (параметры от Engine Speed of Cyl #1 дo #6, Av Engine Speed of All Cyl), отображаемое на дисплее портативного диагностического прибора.

    УКАЗАНИЕ:
    Сначала измеренное значение частоты вращения коленчатого вала двигателя для каждого цилиндра, отображаемое на дисплее диагностического прибора, будет чрезмерно высоким. Приблизительно через 10 с прокручивания коленчатого вала двигателя стартером отображается действительная частота вращения коленчатого вала двигателя для каждого цилиндра.

    ПРИМЕЧАНИЕ:
    1. Испытание "Check the Cylinder Compression" автоматически завершается через 255 с после запуска.
    2. Если испытание Check the Cylinder Compression выключено и коленчатый вал двигателя прокручивается, двигатель запустится.
    3. Если испытание Check the Cylinder Compression необходимо выполнить еще раз после того, как оно было запущено, на портативном диагностическом приборе нажмите EXIT (выход), чтобы возвратиться в окно меню режима Active Test. Затем выполните испытание Check the Cylinder Compression еще раз.
    4. Используйте только полностью заряженную аккумуляторную батарею.
