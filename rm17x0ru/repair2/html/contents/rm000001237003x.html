1. INSTALL REAR CRANKSHAFT OIL SEAL



  1. Apply MP grease to the lip of a new oil seal.

    ПРИМЕЧАНИЕ:
    Keep the lip free from foreign matter.

  1. Using SST, tap in the oil seal until its surface is flush with the engine rear oil seal retainer edge.

    SST
    09223-15030  
    09950-70010   (09951-07100)

    ПРИМЕЧАНИЕ:
    Wipe off any extra grease on the crankshaft.



2. INSTALL FLYWHEEL HOUSING DUST SEAL


3. INSTALL REAR END PLATE

  1. Install the rear end plate with the 2 bolts.

    Момент затяжки:
    27 N*m{ 275 kgf*cm , 20 ft.*lbf }



4. INSTALL FLYWHEEL SUB-ASSEMBLY



  1. Using SST, hold the crankshaft.

    SST
    09213-54015   (91651-60855)
    09330-00021  

  1. Clean the bolts and bolt holes.

  1. Apply adhesive to 2 or 3 threads of each of the bolts.

    Adhesive:
    Toyota Genuine Adhesive 1324, Three Bond 1324 or equivalent

  1. Install the flywheel to the crankshaft.

  1. Temporarily install the flywheel with the 8 bolts.




  1. Tighten the 8 bolts uniformly in several steps in the order shown in the illustration.

    Момент затяжки:
    123 N*m{ 1249 kgf*cm , 90 ft.*lbf }

    ПРИМЕЧАНИЕ:
    Do not start the engine for at least 1 hour after installing.



5. INSTALL CLUTCH DISC ASSEMBLY



  1. Insert SST into the clutch disc. Then insert SST (together with the clutch disc) into the flywheel to install the clutch disc.

    SST
    09301-00110  
    Text in Illustration 

    		Flywheel Side

    ПРИМЕЧАНИЕ:
    Be sure to install the clutch disc so that it is facing in the correct direction.



6. INSTALL CLUTCH COVER ASSEMBLY



  1. Align the matchmarks on the clutch cover and flywheel.
    Text in Illustration 
    *a
    		Matchmark

  1. Tighten the 6 bolts uniformly in the order shown in the illustration, starting with the bolt located near the knock pin on the top.

    SST
    09301-00110  

    Момент затяжки:
    19 N*m{ 194 kgf*cm , 14 ft.*lbf }

    УКАЗАНИЕ:
    Move SST up and down, and right and left lightly after checking that the clutch disc assembly is in the center, and then tighten the bolts.
