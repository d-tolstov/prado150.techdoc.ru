СТОЛЛ-ТЕСТ

УКАЗАНИЕ:
Это испытание предназначено для общей проверки работы двигателя и трансмиссии.

ПРЕДОСТЕРЕЖЕНИЕ:
  1. Для обеспечения безопасности испытание должно выполняться на открытом ровном участке, характеризующемся хорошей силой сцепления.
  2. Столл-тест всегда выполняется, по крайней мере, 2 механиками. Один из них должен следить за состоянием колес и колодок под колесами, пока второй выполняет испытание.

ПРИМЕЧАНИЕ:
Не выполняйте столл-тест дольше 5 с.

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Приведите автомобиль в движение и не останавливайте его, пока температура трансмиссионной жидкости не достигнет 50-80°C (122-176°F).

  3. Дайте двигателю поработать на холостом ходу при выключенной системе кондиционирования.

  4. Установите колодки под все 4 колеса.

  5. Включите стояночный тормоз, а затем нажмите и твердо удерживайте педаль тормоза левой ногой.

  6. Установите рычаг переключения передач в положение D.

  7. Правой ногой до упора выжмите педаль акселератора.

  8. Снимите показания частоты вращения коленчатого вала двигателя (оборотов двигателя при столл-тесте) и сразу же отпустите педаль акселератора.

    Номинальная величина:
    2400 +/- 200 об/мин


    Анализ: 
    Результат испытания
    		Возможная причина
    Обороты двигателя при столл-тесте ниже номинального значения
    1. Нарушена работа муфты свободного хода статора.
    2. Неисправна муфта гидротрансформатора (обороты двигателя при столл-тесте ниже номинального значения на 600 об/мин или более)
    3. Мощности двигателя недостаточно
    Частота вращения коленчатого вала двигателя при столл-тесте выше номинальной
    1. Низкое магистральное давление
    2. Пробуксовка сцепления муфты C1
    3. Нарушение работы муфты свободного хода F3

    ПРИМЕЧАНИЕ:
    Выполняйте данное испытание при температуре ATF от 50 до 80℃ (от 122 до 176°F).


ПРОВЕРКА ЗАПАЗДЫВАНИЯ ПРИ ВКЛЮЧЕНИИ ПЕРЕДАЧИ

УКАЗАНИЕ:
Это испытание предназначено для проверки состояния муфты прямой передачи, муфты переднего хода, тормоза 1-й передачи и тормоза передачи заднего хода.

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Приведите автомобиль в движение и не останавливайте его, пока температура трансмиссионной жидкости не достигнет 50-80°C (122-176°F).

  3. Дайте двигателю поработать на холостом ходу при выключенной системе кондиционирования.

  4. Включите стояночный тормоз, а затем нажмите и твердо удерживайте педаль тормоза.

  5. Проверьте задержку по времени для положения D.

    1. Переведите рычаг переключения передач в положение N и подождите в течение 1 мин.

    2. Переведите рычаг переключения передач в положение D измерьте время, по истечении которого ощущается толчок.

    3. 3 раза повторите оба перечисленных выше действия, а затем рассчитайте среднее время для 3 испытаний.

  6. Проверьте задержку по времени для положения R.

    1. Переведите рычаг переключения передач в положение N и подождите в течение 1 мин.

    2. Переведите рычаг переключения передач в положение R измерьте время, по истечении которого ощущается толчок.

    3. 3 раза повторите 2 процедуры, а затем рассчитайте среднее время для 3 испытаний.

      Номинальная величина:
      Задержка по времени для положения D: менее 1,2 с
      Задержка по времени для положения R: менее 1,5 с


      Анализ: 
      Результат испытания
      		Возможная причина
      Задержка по времени для положения D превышает номинальную величину
      1. Низкое магистральное давление
      2. Износ муфты C1
      3. Нарушение работы муфты свободного хода F3
      Задержка по времени для положения R превышает номинальную величину
      1. Низкое магистральное давление
      2. Износ муфты C3
      3. Износ тормоза B4
      4. Нарушение работы муфты свободного хода F1
