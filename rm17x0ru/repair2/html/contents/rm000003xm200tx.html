ПРОВЕРЬТЕ DTC

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Установите замок зажигания в положение ON (ВКЛ).

  3. Включите портативный диагностический прибор.

  4. Войдите в следующие меню: Chassis / Parking Assist Monitor System / DTC.

  5. Обратитесь к описанию кодов DTC (Нажмите здесь).

    ЭБУ системы помощи при парковке может выдавать коды DTC для указанной ниже системы. УДАЛИТЕ DTC

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Установите замок зажигания в положение ON (ВКЛ).

  3. Включите портативный диагностический прибор.

  4. С помощью портативного диагностического прибора удалите коды из памяти. Войдите в следующие меню: Chassis / Parking Assist Monitor System / DTC.

  5. В соответствии с указаниями на дисплее прибора выберите экран данных кодов неисправностей с помощью кнопки удаления.
