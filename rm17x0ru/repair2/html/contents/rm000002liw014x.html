НАСТРОЙКА ПАРАМЕТРОВ С ПОМОЩЬЮ ПОРТАТИВНОГО ДИАГНОСТИЧЕСКОГО ПРИБОРА (ДЛЯ СПРАВКИ)

УКАЗАНИЕ:
Можно настроить следующие параметры.

ПРИМЕЧАНИЕ:
  1. Если клиент требует изменить настройки, прежде всего следует убедиться, что значения соответствующих параметров могут изменяться.
  2. Прежде чем приступать к настройке параметров, запишите текущие установки.

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Установите замок зажигания в положение ON (ВКЛ).

  3. Включите портативный диагностический прибор.

  4. Войдите в следующие меню: Body / Air Conditioner / Utility / Customize.

  5. Проверьте состояние в соответствии с таблицей ниже.

    Air Conditioner 
    Индикация
    		Значение по умолчанию
    		Содержание
    		Настройка
    Set Temperature Shift
    		Normal
    		Эта функция позволяет задавать отклонение температуры.
    		+2 C, +1 C, Normal, -1 C или -2 C
    Compressor Mode
    		Automatic
    		Эта функция используется для автоматического включения системы кондиционирования нажатием кнопки AUTO, если вентилятор включен, а система кондиционирования выключена.
    		Automatic или Manual
    Air Inlet Mode
    		Automatic
    		Эта функция позволяет автоматически изменять режим на режим рециркуляции после включения системы кондиционирования.
    		Automatic или Manual
    Foot / DEF Auto Mode
    		ON (ВКЛ)
    		Эта функция обеспечивает автоматическую подачу воздуха через воздуховоды с дефлектором в передней и задней выемках для ног и передний оттаиватель в автоматическом режиме.
    		ON (ВКЛ) или OFF (ВЫКЛ)
    Foot/DEF Automatic Blow Up Function
    		ON (ВКЛ)
    		Эта функция вызывает автоматическое переключение скорости вентилятора при включенном переднем оттаивателе.
    		ON (ВКЛ) или OFF (ВЫКЛ)
    Add Foot Air in FACE Mode
    		ON (ВКЛ)
    		Когда выбирается режим FACE, в углубления для ног подается дополнительное количество воздуха.
    		ON (ВКЛ) или OFF (ВЫКЛ)
