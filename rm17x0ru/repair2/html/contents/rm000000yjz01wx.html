1. INSTALL IGNITION SWITCH ASSEMBLY

  1. Install the ignition switch with the 2 screws.

  1. Connect the ignition switch connector and attach the wire harness clamp.



2. INSTALL LOWER STEERING COLUMN COVER

  1. Attach the 2 claws to install the lower steering column cover.

  1. Install the 2 screws.

    УКАЗАНИЕ:
    Turn the steering wheel to the right and left as necessary to install the 2 screws.
