ОПИСАНИЕ

ЭБУ предупреждения о недопустимой дистанции*1 или ЭБУ системы помощи при парковке*2 получает сигнал заднего хода от датчика положения паркинга/нейтрали*3 или выключателя света заднего хода*4.
  1. *1: Для моделей без системы помощи при парковке и/или системы монитора бокового обзора
  2. *2: Для моделей с мониторной системой помощи при парковке и/или мониторной системой бокового обзора
  3. *3: Для моделей с автоматической трансмиссией
  4. *4: Для моделей с механической трансмиссией
