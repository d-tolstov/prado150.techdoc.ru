1. INSTALL DELIVERY VALVE HOLDER



  1. Install 4 new gaskets and the 4 valves to the distributive head.

  1. Install the 4 springs to the 4 delivery valve holders.

  1. Using SST, install the delivery holders.

    SST
    09260-54012   (09269-54020)

    Момент затяжки:
    59 N*m{ 600 kgf*cm , 43 ft.*lbf }



2. INSTALL DISTRIBUTIVE HEAD PLUG



  1. Install a new O-ring to the distributive head plug.

  1. Using SST, install the head plug.

    SST
    09260-54012   (09262-54010)

    Момент затяжки:
    88 N*m{ 900 kgf*cm , 65 ft.*lbf }



3. INSTALL TIMING CONTROL VALVE



  1. Using a 5 mm hexagon wrench, install the timing control valve with the 2 bolts.
    Text in Illustration 
    *1
    		5 mm Hexagon Wrench



4. INSTALL FUEL FILTER TO INJECTION PUMP FUEL PIPE



  1. Install a new washer and the hollow screw.

    Момент затяжки:
    37 N*m{ 375 kgf*cm , 27 ft.*lbf }

  1. Install 2 new washers and the fuel pipe with the cap nut.

    Момент затяжки:
    27 N*m{ 271 kgf*cm , 20 ft.*lbf }

  1. Install the wiring bracket with the bolt.

    Момент затяжки:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }



5. INSTALL NO. 4 NOZZLE LEAKAGE PIPE

  1. Install 2 new washers and the nozzle leakage pipe with the overflow screw.

    Момент затяжки:
    27 N*m{ 271 kgf*cm , 20 ft.*lbf }
