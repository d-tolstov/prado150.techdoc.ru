для подготовки Нажмите здесь


ОПИСАНИЕ

Когда зажигание включено, центральный блок управления системы SRS по шине CAN передает в щиток приборов сигналы состояния замка ремня безопасности левого*1 (или правого*2) переднего сиденья. Если ремень безопасности водителя не пристегнут, щиток приборов включает контрольную лампу ремня безопасности водителя в мигающем режиме. Если ремень безопасности пристегнут, контрольная лампа не включается.
  1. *1: Для моделей с левосторонним рулевым управлением
  2. *2: Для моделей с правосторонним рулевым управлением
