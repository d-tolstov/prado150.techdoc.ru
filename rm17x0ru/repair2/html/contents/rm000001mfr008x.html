DTC    32 (1)    Injection Pump System Malfunction

для подготовки Нажмите здесь


DESCRIPTION

To compensate for the change in injection volume and injection timing caused by the variance in the injection pump itself, a correction is made by using the data that is stored in the ROM in the fuel pump correction unit.

DTC No.
DTC Detection Condition
Trouble Area
32 (1)
  1. When the ignition switch is turned to ON, or when the ECM determines if there is an error in the RAM of the ECM, the ECM conducts single communication. If one of the following conditions occurs in single communication, the ECM conducts another communication. While the ECM is conducting communication, the ECM stores DTC 32 (1) if (a) occurs 20 times continuously, (b) occurs 20 times continuously, or (c) occurs 2 times continuously.
  2. (a) There was a communication error when receiving data.
  3. (b) Data sent from the injection pump correction unit does not match the data received in the ECM.
  4. (c) Data received from the pump ROM is not input for 1024 msec.

    УКАЗАНИЕ:
    Even when determining if the process is completed normally, if an undefined code is found while checking the identification code of data received, the ECM conducts communication again. If an undefined code occurs 20 times continuously, the ECM stores DTC 32 (1).

  1. Open or short in injection pump correction unit circuit
  2. Injection pump correction unit
  3. ECM
