1. INSTALL FUEL TANK BREATHER HOSE

  1. Install the fuel tank breather hose to the fuel sub tank.



2. INSTALL FUEL TANK TO FILLER PIPE HOSE

  1. Install the fuel tank to filler pipe hose to the fuel sub tank.



3. INSTALL FUEL TANK VENT TUBE ASSEMBLY

  1. Install a new gasket and fuel tank vent tube with the 8 screws.

    Момент затяжки:
    4.0 N*m{ 41 kgf*cm , 35 in.*lbf }

  1. Attach the wire harness clamp.



4. INSTALL FUEL SUB TANK SUB-ASSEMBLY

  1. Set the fuel sub tank on a transmission jack and lift up the transmission jack.

  1. Connect the 2 fuel tank bands with the 4 bolts.

    Момент затяжки:
    40 N*m{ 408 kgf*cm , 30 ft.*lbf }

  1. Connect the fuel sender gauge connector.



5. CONNECT FUEL TANK BREATHER HOSE

  1. Connect the fuel tank breather hose to the filler pipe.



6. CONNECT FUEL TANK TO FILLER PIPE HOSE

  1. Connect the fuel tank to filler pipe hose to the filler pipe.



7. CONNECT FUEL HOSE

  1. Connect the fuel evaporation hose, fuel breather hose and 2 fuel hoses.



8. INSTALL NO. 2 FUEL TANK PROTECTOR

  1. Install the No. 2 fuel tank protector with the 5 bolts.

    Момент затяжки:
    20 N*m{ 204 kgf*cm , 15 ft.*lbf }



9. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

ПРИМЕЧАНИЕ:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (See page Нажмите здесь).



10. BLEED AIR FROM FUEL SYSTEM



  1. Using the hand pump mounted on the fuel filter cap, bleed the air from the fuel system. Continue pumping until the pump resistance increases.



11. INSPECT FOR FUEL LEAK

  1. Check that there are no fuel leaks anywhere in the fuel system after performing maintenance.

    УКАЗАНИЕ:
    When checking for fuel leaks, make sure that there is pressure in the fuel line.
