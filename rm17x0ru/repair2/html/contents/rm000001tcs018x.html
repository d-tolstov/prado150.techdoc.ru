СЧИТАЙТЕ ДАННЫЕ В РЕЖИМЕ DATA LIST

УКАЗАНИЕ:
В режиме Data List на дисплей портативного диагностического прибора можно вывести данные о состоянии переключателей, датчиков, приводов и пр. без снятия деталей с автомобиля. Такая внешняя проверка может оказаться очень полезной, поскольку позволяет выявлять нерегулярно возникающие состояния и сигналы, не нарушая монтажа деталей и электропроводки. Считывание информации в режиме Data List на первом этапе поиска неисправностей позволяет сократить время, затрачиваемое на диагностику.

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Установите замок зажигания в положение ON (ВКЛ).

  3. Включите портативный диагностический прибор.

  4. Войдите в следующие меню: Body/Immobiliser/Data List.

  5. В соответствии с указаниями на дисплее портативного диагностического прибора считайте данные в режиме Data List.

    Immobiliser 
    Информация на дисплее прибора
    		Измеряемая величина / диапазон измерения
    		Нормальное состояние
    		Замечание по диагностике
    D Door Courtesy SW*1
    		Выключатель освещения проема двери водителя / ON (ВКЛ) или OFF (ВЫКЛ)
    		ON (ВКЛ): Дверь со стороны водителя открыта
    OFF (ВЫКЛ): дверь водителя закрыта
    		-
    Key SW
    		Сигнал выключателя разблокировки ключа / ON (ВКЛ) или OFF (ВЫКЛ)
    		ON (ВКЛ): Ключ вставлен в замок зажигания
    OFF (ВЫКЛ): Ключ не вставлен в замок зажигания
    		-
    IG SW
    		Сигнал замка зажигания / ON (ВКЛ) или OFF (ВЫКЛ)
    		ON (ВКЛ): Зажигание включено
    OFF (ВЫКЛ): Замок зажигания в положении ACC или OFF (ВЫКЛ)
    		-
    Immobiliser
    		Состояние системы иммобилайзера / Set (вкл) или Unset (выкл)
    		Set (установлено): Ключ не вставлен в замок зажигания
    Unset (не установлено): Ключ вставлен в замок зажигания
    		-
    Permit (Start)*2
    		Сигнал разрешения запуска / OK или NG
    		OK: запуск двигателя разрешен
    NG: запуск двигателя не разрешен
    		-
    Response
    		Данные микросхемы электронного ключа зажигания / OK или NG
    		ОК: Данные OK
    NG: Ошибка данных
    		-
    Frame Error
    		Данные микросхемы электронного ключа зажигания / OK или NG
    		ОК: Данные OK
    NG: Ошибка данных
    		-
    Different Serial Number
    		Данные микросхемы электронного ключа зажигания / OK или NG
    		ОК: Данные OK
    NG: Ошибка данных
    		-
    Different Encrypt Code
    		Данные микросхемы электронного ключа зажигания / OK или NG
    		ОК: Данные OK
    NG: Ошибка данных
    		-
    Abnormal Status
    		Данные микросхемы электронного ключа зажигания / OK или NG
    		ОК: Данные OK
    NG: Ошибка данных
    		-
    BCC Malfunction
    		Сигнал электронной микросхемы ключа / OK или NG
    		ОК: отправлены верные данные
    NG: отправлены неверные данные
    		-
    Sub Key
    		Сигнал кода дополнительного ключа / Reg или Unreg
    		Reg: Отправлен зарегистрированный код дополнительного ключа
    Unreg: Отправлен неверный код дополнительного ключа
    		-
    Master Key
    		Сигнал кода главного ключа/ Reg или Unreg
    		Reg: Отправлен зарегистрированный код главного ключа
    Unreg: Отправлен неверный код главного ключа
    		-
    Transponder S-code
    		Число зарегистрированных дополнительных ключей / мин.: 0, макс.: 3
    		Число зарегистрированных дополнительных ключей
    		-
    Transponder M-code
    		Число зарегистрированных главных ключей / мин.: 0, макс.: 5
    		Число зарегистрированных главных ключей
    		-
    Reg Code Space Full
    		Память для регистрации кода ключа переполнена / Yes или No
    		No: Можно регистрировать дополнительные коды ключей
    Yes: Нельзя регистрировать дополнительные коды ключей
    		-
    +B
    		Определение +B / Normal или Break
    		Normal: Питание от аккумуляторной батареи в норме
    Break: Неисправность питания от аккумуляторной батареи
    		-
    Antenna Coil Status
    		Состояние антенной катушки / Normal или Fail
    		Normal: Антенная катушка в порядке
    Fail: Антенная катушка неисправна
    		-
    EEPROM Status*1
    		Состояние ЭСППЗУ / OK или NG
    		ОК: Данные OK
    NG: Ошибка данных
    		-
    E/G Start Permission*1
    		Разрешение запуска двигателя / OK или NG
    		OK: запуск двигателя разрешен
    NG: запуск двигателя не разрешен
    		-
    Immobi ECU Comm Spd*1
    		Скорость обмена данными ЭБУ иммобилайзера / Unknown, 50bps, 200bps или No Res
    		Unknown: Скорость обмена данными ЭБУ неизвестна
    50bps: Скорость обмена данными ЭБУ 50 бит/с
    200bps: Скорость обмена данными ЭБУ 200 бит/с
    No Res: Нет ответа на запрос о скорости обмена данными ЭБУ
    		-
    The Number of DTCs*1
    		Количество кодов DTC / мин.: 0, макс.: 255
    		Количество зарегистрированных кодов DTC
    		-

    1. *1: для моделей с 2TR-FE
    2. *2: для моделей с 5L-E


ВЫПОЛНИТЕ ПРОВЕРКУ В РЕЖИМЕ ACTIVE TEST

УКАЗАНИЕ:
Выполнение активной диагностики с использованием портативного диагностического прибора дает возможность управлять реле, электровакуумными клапанами, приводами и т.д. без снятия деталей с автомобиля. Такая внешняя функциональная проверка может оказаться очень полезной, поскольку позволяет выявлять эпизодические неисправности, не нарушая монтаж деталей и электропроводки. Выполнение активной диагностики на начальном этапе поиска неисправностей позволяет сократить время, затрачиваемое на диагностику. В режиме Active Test на дисплей диагностического прибора можно вывести список параметров Data list.

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Установите замок зажигания в положение ON (ВКЛ).

  3. Включите портативный диагностический прибор.

  4. Войдите в следующие меню: Body/Immobiliser/Active Test.

  5. В соответствии с указаниями на дисплее портативного диагностического прибора выполните диагностику в режиме Active Test.

    Immobiliser 
    Информация на дисплее прибора
    		Испытываемое устройство
    		Диапазон регулирования
    		Замечание по диагностике
    Security Indicator
    		Контрольная лампа безопасности
    		ON (ВКЛ) / OFF (ВЫКЛ)
    		Испытание возможно при выполнении следующих условий:
    1. Ключ вставлен в замок зажигания.
