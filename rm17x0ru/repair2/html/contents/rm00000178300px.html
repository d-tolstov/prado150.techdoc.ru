3. INSTALL NO. 1 TRANSMISSION CLUTCH HUB



  1. Apply a light coat of gear oil to the sleeve and hub.

  1. Install the clutch hub to the clutch hub sleeve.
    Text in Illustration 
    *a
    		1st Gear Side
    *b
    		2nd Gear Side

  1. Install one of the balls to each shifting key.

  1. Install one of the springs to each shifting key.

  1. Install the shifting keys with balls and springs to the No. 1 clutch hub.

    ПРИМЕЧАНИЕ:
    Be careful to prevent the balls from flying off.



4. INSTALL 2ND GEAR NEEDLE ROLLER BEARING



  1. Coat the 2nd gear needle roller bearing with gear oil, and then install it to the counter gear.



5. INSTALL COUNTER SHAFT 2ND SPEED GEAR



  1. Coat the counter shaft 2nd speed gear with gear oil, and then install it to the counter gear.

  1. Coat the No. 1 synchronizer ring set with gear oil, and then install it to the counter gear.

  1. Using SST and a press, install the No. 1 transmission clutch hub to the counter gear.

    SST
    09316-60011  
    Text in Illustration 
    *a
    		Groove
    *b
    		Protrusion

    УКАЗАНИЕ:
    Align the groove of the clutch hub with the protrusion of the synchronizer ring.

  1. Check that the gear and synchronizer ring move smoothly.

13. INSTALL COUNTERSHAFT REVERSE GEAR



  1. Install the key spring and 2 shifting keys to the reverse gear.
    Text in Illustration 
    *1
    		Reverse Gear
    *2
    		Hub Sleeve
    *3
    		Shifting Key
    *4
    		Key Spring

    УКАЗАНИЕ:
    1. Install each shifting key with its groove on the reverse gear side.
    2. Install the key spring with its claw on the reverse gear side.
    3. Refer to the illustration when installing the key spring.

  1. Install the hub sleeve to the reverse gear.

  1. Coat the countershaft reverse gear with gear oil, and then install it to the counter gear.



14. INSTALL REVERSE GEAR BEARING RACE INNER



  1. Align the groove of the reverse gear bearing race inner with the ball and install the reverse gear bearing race inner.

17. INSPECT REVERSE GEAR THRUST CLEARANCE



  1. Using a feeler gauge, measure the reverse gear thrust clearance.

    Standard clearance:
    0.125 to 0.375 mm (0.00493 to 0.0148 in.)

    If the clearance is outside the specification, replace the gear, spline piece or bearing race.

    УКАЗАНИЕ:
    Replace the part or parts determined to be the most likely cause of the problem.



18. INSPECT 1ST GEAR THRUST CLEARANCE



  1. Using a dial indicator, measure the 1st gear thrust clearance.

    Standard clearance:
    0.10 to 0.43 mm (0.00394 to 0.0169 in.)

    If the clearance is outside the specification, replace the gear, clutch hub, spline piece or counter gear.

    УКАЗАНИЕ:
    Replace the part or parts determined to be the most likely cause of the problem.



19. INSPECT 2ND GEAR THRUST CLEARANCE



  1. Using a dial indicator, measure the 2nd gear thrust clearance.

    Standard clearance:
    0.10 to 0.43 mm (0.00394 to 0.0169 in.)

    If the clearance is outside the specification, replace the gear, clutch hub or counter gear.

    УКАЗАНИЕ:
    Replace the part or parts determined to be the most likely cause of the problem.



20. INSPECT REVERSE GEAR RADIAL CLEARANCE



  1. Using a dial indicator, measure the reverse gear radial clearance.

    Standard clearance:
    0.015 to 0.065 mm (0.000591 to 0.00255 in.)

    If the clearance is outside the specification, replace the reverse gear needle roller bearing or shaft.

    УКАЗАНИЕ:
    Replace the part or parts determined to be the most likely cause of the problem.
