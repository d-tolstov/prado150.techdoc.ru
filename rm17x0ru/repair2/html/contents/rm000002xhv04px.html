DTC    B2785    Ошибка обмена данными между ЭБУ шины LIN

для подготовки Нажмите здесь


ОПИСАНИЕ

ЭБУ сертификации периодически контролирует шину передачи данных LIN между устройствами, связанными с сертификацией. Код DTC B2785 регистрируется при обнаружении 3 последовательных нарушений обмена данными по шине LIN между устройствами, связанными с сертификацией.

Код DTC
Условие обнаружения DTC
Неисправный участок
B2785
ЭБУ сертификации 3 раза подряд обнаруживает нарушение обмена данными по шине LIN между устройствами, связанными с сертификацией.
  1. ЭБУ сертификации
  2. ЭБУ распределения питания
  3. Блок идентификационного кода
  4. Привод блокировки рулевого управления (ЭБУ блокировки рулевого управления)
  5. Жгут или разъем

СХЕМА СОЕДИНЕНИЙ




ПОСЛЕДОВАТЕЛЬНОСТЬ ПРОВЕРКИ

ПРИМЕЧАНИЕ:
При использовании портативного диагностического прибора для диагностики с выключенным зажиганием:
Подключите портативный диагностический прибор к автомобилю и с интервалом в 1,5 секунды включайте и выключайте освещение дверного проема до тех пор, пока не установится связь между диагностическим прибором и автомобилем.

УКАЗАНИЕ:
Если одновременно выводятся диагностические коды неисправности DTC B2785 и B2287, в первую очередь выполните поиск неисправностей для DTC B2785.
