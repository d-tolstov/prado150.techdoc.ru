УКАЗАНИЕ:
  1. Данная процедура поиска неисправностей предназначена для выявления причин неравномерного холостого хода и чрезмерной вибрации двигателя.
  2. Значения, указанные в блок-схеме поиска неисправностей, приводятся только для справки. Значения в режиме Data List могут немного изменяться в зависимости от срока службы автомобиля или условий измерения. Если значения в режиме Data List находятся в пределах допустимого диапазона, это не означает, что неисправности отсутствуют. Возможно, имеет место скрытая неисправность.
  3. Перед выполнением проверки убедитесь, что в конструкцию автомобиля не вносились какие-либо изменения.


Неисправности и признаки для узлов топливной системы Common Rail для дизельного двигателя

  1. Система управления двигателем

    Датчик массового расхода воздуха воздуха 
    Основная неисправность
    		Понижение рабочих характеристик (попадание постороннего объекта)
    Признаки
    		Недостаточная мощность, черный дым
    Data List
    		MAF

    УКАЗАНИЕ:
    Максимальный объем впрыска топлива регулируется в соответствии с выходным сигналом датчика массового расхода воздуха.


    Система свечей накаливания 
    Основная неисправность
    		Обрыв цепи, неисправность реле свечей накаливания
    Признаки
    		Затрудненный запуск, неравномерный холостой ход, детонация, белый дым (в холодную погоду)
    Параметры из Data List
    		Проверьте контрольную лампу свечи накаливания
    Способ диагностики
    		Измерьте сопротивление свечи накаливания

    Двигатель 
    Основная неисправность
    		Потеря компрессии
    Признаки
    		Неравномерный холостой ход (постоянно недостаток мощности)
    Data List
    		Engine Speed of Cyl
    1. Если при прокручивании коленчатого вала двигателя во время испытания Active Test "Check the Cylinder Compression" частота вращения коленчатого вала двигателя для одного цилиндра примерно на 100 об/мин выше, чем для других цилиндров, в данном цилиндре возможна потеря компрессии.
    Injection Feedback Val
    1. Если значение Injector Feedback Val превышает 3 мм3/ход, возможно, в соответствующем цилиндре имеется неисправность.

  2. Система впрыска дизельного двигателя

    Нагнетающий топливный насос в сборе 
    Основная неисправность
    		-
    Признаки
    		Затрудненный запуск, заглохание двигателя, неравномерный холостой ход, недостаточная мощность
    Параметры из Data List
    		Fuel Press, Target Common Rail Pressure, Target Pump SCV Current
    1. При стабильных условиях (например, на холостом ходу) значение Fuel Press находится в пределах интервала "Target Common Rail Pressure" +/-5000 кПа.
    2. Если давление в топливной системе на 20000 кПа ниже заданного давления, будет ощущаться недостаток мощности.
    3. Если давление в топливной системе ниже 25000 кПа, холостой ход будет неравномерным.

    УКАЗАНИЕ:
    1. Давление в топливной системе при запуске двигателя изменяется, однако при запуске прогретого двигателя составляет примерно 25000 кПа.
    2. Если значение Target Pump SCV Current составляет 3000 мА или выше, клапан регулирования всасывания может заедать.

    Диагностический код неисправности DTC
    		Даже если значение Fuel Press меньше значения Target Common Rail Pressure, DTC не сохраняется.

    Фильтрующий элемент топливного фильтра в сборе 
    Основная неисправность
    		Засорение
    Признаки
    		Затрудненный запуск, заглохание двигателя, неравномерный холостой ход, недостаточная мощность
    Параметры из Data List
    		Fuel Press, Target Common Rail Pressure
    1. При стабильных условиях (например, на холостом ходу) давление в топливной системе находится в пределах интервала "Target Common Rail Pressure" +/-5000 кПа.
    2. Если давление в топливной системе на 20000 кПа ниже заданного давления, будет ощущаться недостаток мощности.
    3. Если давление в топливной системе ниже 25000 кПа, холостой ход будет неравномерным.

    УКАЗАНИЕ:
    Давление в топливной системе при запуске двигателя изменяется, однако при запуске прогретого двигателя составляет примерно 25000 кПа.

    Диагностический код неисправности DTC
    		Даже если значение Fuel Press меньше значения Target Common Rail Pressure, DTC не сохраняется.

    Форсунка в сборе 
    Основная неисправность
    		Засорение
    Признаки
    		Неравномерный холостой ход, недостаточная мощность, черный дым, белый дым, детонация
    Параметры из Data List
    		Injection Feedback Val
    1. Если значение Injector Feedback Val превышает 3 мм3/ход, возможно, в соответствующем цилиндре имеется неисправность. Это значение считывается после работы прогретого двигателя на холостом ходу в течение 1 мин (когда температура охлаждающей жидкости двигателя превышает 70°C (158°F)).

    Электронный блок привода форсунок 
    Основная неисправность
    		Неисправность цепи: топливная форсунка в сборе не открывается.
    Признаки
    		Затрудненный запуск, неравномерный холостой ход, недостаток мощности, черный дым, белый дым, детонация
    Параметры из Data List
    		То же, что для форсунки в сборе
    Диагностический код неисправности DTC
    		Когда электронный блок привода форсунок неисправен, могут сохраняться некоторые DTC.

    Датчик давления топлива 
    Основная неисправность
    		Обрыв цепи, понижение рабочих характеристик (попадание постороннего объекта)
    Признаки
    		Затрудненный запуск, неравномерный холостой ход, заглохание двигателя, недостаток мощности
    Параметры из Data List
    		Fuel Press, Target Common Rail Pressure
    1. В стоящем на месте автомобиле медленно повысьте частоту вращения коленчатого вала двигателя с частоты холостого хода до 3000 об/мин и убедитесь, что изменение значения Fuel Press повторяет изменение значения Target Common Rail Pressure. Если датчик давления топлива неисправен, фактическое давление в топливной системе может отклоняться от заданного (при этом значение может не отклоняться даже при наличии неисправности).
    Диагностический код неисправности DTC
    		Если датчик давления в топливной системе неисправен, могут регистрироваться некоторые DTC.

    Неравномерная подача топлива 
    Основная неисправность
    		-
    Признаки
    		Затрудненный запуск, неравномерный холостой ход (особенно в холодную погоду)

  3. Система рециркуляции отработавших газов дизельного двигателя

    Система РОГ 
    Основная неисправность
    1. Не перемещается плавно
    2. Не полностью закрывается
    Признаки
    1. Неравномерный холостой ход
    2. Заедание клапана РОГ в закрытом положении: громкий звук работы турбонагнетателя.
    3. Заедание клапана РОГ в открытом положении: затрудненный запуск (двигатель не глохнет), черный дым, недостаток мощности (если при большой нагрузке объем рециркулирующих отработавших газов избыточен, то при трогании автомобиля будет ощущаться недостаток мощности).
    Параметры из Data List
    		Actual EGR Valve Pos., Target EGR Pos.
    1. Как правило, Actual EGR Valve Pos. = Target EGR Pos. +/-5% (полностью закрытое положение: 0%, полностью открытое положение: 100%).
    2. Выполните испытание клапана РОГ в режиме Active Test и убедитесь, что изменение значения Actual EGR Valve Pos. повторяет изменение значения Target EGR Pos. (при возникновении неисправности следует учитывать температуру охлаждающей жидкости двигателя и температуру воздуха на впуске).
    3. Когда выключатель зажигания находится в состоянии ON (ВКЛ) (IG) (двигатель остановлен), клапан РОГ полностью закрыт.
    4. На холостом ходу после прогрева двигателя клапан РОГ открывается наполовину.
    EGR Close Lrn. Val., EGR Close Lrn. Status
    1. В режиме холостого хода, когда параметр EGR Close Lrn. Status имеет значение OK, номинальный диапазон значений EGR Close Lrn. Val. составляет 0-1 В.
    2. В случаях, когда EGR Close Lrn. Status. имеет значение NG, либо значение EGR Close Lrn. Val. выходит за пределы номинального диапазона (0-1 В), вероятно, клапан РОГ не может закрыться полностью.

  4. Дроссельная заслонка дизельного двигателя

    Система дроссельной заслонки дизельного двигателя 
    Основная неисправность
    		Заедание, неплавное перемещение
    Признаки
    1. Заедание в закрытом положении: недостаток мощности, затрудненный запуск, неравномерный холостой ход, заглохание двигателя, черный дым. Это может происходить при заедании в почти полностью закрытом положении.
    2. Заедание в открытом положении: усиливается шум, производимый турбонагнетателем. При остановке двигателя может возникать вибрация двигателя.
    Параметры из Data List
    1. Actual Throttle Position
      0%: полностью открыта
      100%: полностью закрыта
    2. Когда выключатель зажигания находится в состоянии ON (ВКЛ) (IG) (двигатель остановлен), дроссельная заслонка дизельного двигателя полностью открыта. На холостом ходу дроссельная заслонка дизельного двигателя открывается наполовину. При переводе выключателя зажигания из состояния ON (ВКЛ) (IG) в состояние OFF (ВЫКЛ) дроссельная заслонка временно полностью закрывается.

4.ЗАФИКСИРУЙТЕ МГНОВЕННЫЕ ЗНАЧЕНИЯ ПАРАМЕТРОВ В РЕЖИМЕ ХОЛОСТОГО ХОДА (ПРОЦЕДУРА 4)



  1. Подсоедините портативный диагностический прибор к DLC3.
