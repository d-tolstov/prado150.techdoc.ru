DTC    U0100    Нарушение связи с ECM

DTC    U0142    Прекращение обмена данными с главным ЭБУ кузова

DTC    U0155    Нарушение связи с щитком приборов

для подготовки Нажмите здесь


ОПИСАНИЕ

Блок управления системой кондиционирования обменивается данными с ECM, главным ЭБУ кузова (бортовым ЭБУ сети мультиплексной связи) и щитком приборов через мультиплексную шину CAN.

Код DTC
Условие обнаружения DTC
Неисправный участок
U0100
Отсутствие связи с ECM
  1. Система передачи данных CAN
  2. ECM
U0142
Отсутствие связи с главным ЭБУ кузова.
  1. Система передачи данных CAN
  2. Главный ЭБУ кузова (бортовой ЭБУ сети мультиплексной связи)
U0155
Отсутствие связи с щитком приборов
  1. Система передачи данных CAN
  2. Щиток приборов в сборе

ПОСЛЕДОВАТЕЛЬНОСТЬ ПРОВЕРКИ

1.ПРОВЕРЬТЕ DTC (СИСТЕМЫ ПЕРЕДАЧИ ДАННЫХ CAN)

  1. Проверьте работоспособность мультиплексной шины CAN с помощью портативного диагностического прибора.
