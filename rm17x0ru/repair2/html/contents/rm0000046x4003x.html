УКАЗАНИЕ:
При установке элемента защиты наружной облицовки двери багажного отделения нагрейте нижний кронштейн опоры заднего номерного знака и поверхность декоративного элемента двери багажного отделения с помощью нагревательной лампы.

Номинальное значение / Номинальный режим:
Параметр / Устройство
Температура
Нижний кронштейн опоры заднего номерного знака
20–30°C (68–86°F)
Декоративный элемент двери багажного отделения
20–30°C (68–86°F)

ПРИМЕЧАНИЕ:
Не допускайте чрезмерного разогрева наружной накладки двери багажного отделения.



1. УСТАНОВИТЕ ЭЛЕМЕНТ ЗАЩИТЫ НАРУЖНОЙ НАКЛАДКИ ДВЕРИ БАГАЖНОГО ОТДЕЛЕНИЯ

  1. для нижнего кронштейна опоры заднего номерного знака:

    1. При использовании нового нижнего кронштейна опоры заднего номерного знака:

      1. Очистите поверхность нижнего кронштейна опоры заднего номерного знака.
      2. С помощью нагревательной лампы нагрейте поверхность нижнего кронштейна опоры заднего номерного знака.

    2. При повторном использовании нижнего кронштейна опоры заднего номерного знака:

      1. С помощью нагревательной лампы нагрейте поверхность нижнего кронштейна опоры заднего номерного знака.
      2. Удалите двухстороннюю клейкую ленту с нижнего кронштейна опоры заднего номерного знака.
      3. Удалите остатки клейкой ленты чистящим средством.

    3. Установите элемент защиты наружной облицовки двери багажного отделения.

  1. для колпака лампы освещения заднего номерного знака:

    1. При использовании нового декоративного элемента двери багажного отделения:

      1. Очистите тыльную поверхность декоративного элемента двери багажного отделения.
      2. С помощью нагревательной лампы разогрейте поверхность декоративного элемента двери багажного отделения.

    2. При повторном использовании декоративного элемента двери багажного отделения:

      1. С помощью нагревательной лампы разогрейте поверхность декоративного элемента двери багажного отделения.
      2. Удалите двухстороннюю клейкую ленту с декоративного элемента двери багажного отделения.
      3. Удалите остатки клейкой ленты чистящим средством.

    3. Установите элемент защиты наружной облицовки двери багажного отделения.



2. УСТАНОВИТЕ ДЕКОРАТИВНЫЙ ЭЛЕМЕНТ ДВЕРИ БАГАЖНОГО ОТДЕЛЕНИЯ

  1. Введите в зацепление 6 захватов, чтобы установить декоративный элемент двери багажного отделения.



3. УСТАНОВИТЕ ВЫКЛЮЧАТЕЛЬ ЭЛЕКТРОННОГО КЛЮЧА БАГАЖНОГО ОТДЕЛЕНИЯ (для моделей с системой посадки и запуска)



  1. Введите в зацепление 2 направляющих, чтобы установить выключатель электронного ключа на двери багажного отделения.

  1. Заверните 2 винта.



4. УСТАНОВИТЕ ЛАМПУ ОСВЕЩЕНИЯ НОМЕРНОГО ЗНАКА В СБОРЕ

  1. Закрепите лампу освещения номерного знака 4 винтами.



5. УСТАНОВИТЕ КОЛПАК ЛАМПЫ ОСВЕЩЕНИЯ ЗАДНЕГО НОМЕРНОГО ЗНАКА

  1. Установите колпак лампы освещения заднего номерного знака и закрепите его 5 винтами.



6. УСТАНОВИТЕ ШНУР ЛАМПЫ ОСВЕЩЕНИЯ НОМЕРНОГО ЗНАКА (для моделей без системы посадки и запуска)

  1. Подсоедините разъем, чтобы установить шнур лампы освещения номерного знака.
