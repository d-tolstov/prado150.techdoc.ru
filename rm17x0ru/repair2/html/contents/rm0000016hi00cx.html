1. INSTALL GLOW PLUG ASSEMBLY

  1. Using a 12 mm deep socket wrench, install the 4 glow plugs.

    Момент затяжки:
    13 N*m{ 130 kgf*cm , 10 ft.*lbf }



2. INSTALL NO. 1 GLOW PLUG CONNECTOR

  1. Install the No. 1 glow plug resistor insulator and No. 1 glow plug connector.




  1. Install the glow plug connector with the 4 nuts. Uniformly tighten the nuts.

    Момент затяжки:
    1.0 N*m{ 10 kgf*cm , 9 in.*lbf }

    Text in Illustration 
    *1
    		Nut
    *2
    		Washer
    *3
    		No. 2 Glow Plug Resistor Insulator
    *4
    		Engine Wire
    *5
    		No. 1 Glow Plug Connector
    *6
    		No. 1 Glow Plug Resistor Insulator
    *7
    		Bolt

  1. Install the 4 screw grommets.

  1. Connect the engine wire and install the No. 2 glow plug resistor insulator and washer with the bolt.

    Момент затяжки:
    5.0 N*m{ 51 kgf*cm , 44 in.*lbf }



3. CONNECT VENTILATION HOSE

  1. Connect the ventilation hose.



4. INSTALL INTAKE PIPE

  1. Install the intake pipe with the 2 bolts and tighten the hose clamp.

    Момент затяжки:
    for bolt:
    18 N*m{ 184 kgf*cm , 13 ft.*lbf }
    for clamp:
    6.0 N*m{ 61 kgf*cm , 53 in.*lbf }



5. INSTALL RESONATOR WITH AIR CLEANER CAP SUB-ASSEMBLY

  1. Вставьте петли крышки воздушного фильтра и шланг в корпус воздушного фильтра, а затем закрепите 4 откидных защелки.

  1. Установите крышку воздушного фильтра и закрепите ее зажимом.

    Момент затяжки:
    5,0 N*m{ 51 kgf*cm , 44 in.*lbf }

  1. Закрепите зажим жгута проводов.

  1. Подсоедините 2 зажима и разъем.



6. CONNECT CABLE TO NEGATIVE BATTERY TERMINAL

ПРИМЕЧАНИЕ:
When disconnecting the cable, some systems need to be initialized after the cable is reconnected (See page Нажмите здесь).
