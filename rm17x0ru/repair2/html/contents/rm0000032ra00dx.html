1. INSTALL CLUTCH RELEASE CYLINDER ASSEMBLY

  1. Install the clutch release cylinder with the 2 bolts.

    Момент затяжки:
    12 N*m{ 120 kgf*cm , 9 ft.*lbf }



2. CONNECT CLUTCH RELEASE CYLINDER TO FLEXIBLE HOSE TUBE

  1. Using a union nut wrench, connect the tube to the clutch release cylinder.

    Момент затяжки:
    15 N*m{ 155 kgf*cm , 11 ft.*lbf }

    ПРИМЕЧАНИЕ:
    Use the formula to calculate special torque values for situations where a union nut wrench is combined with a torque wrench (See page Нажмите здесь).



3. INSTALL FRONT PROPELLER SHAFT ASSEMBLY

  1. Install the front propeller shaft (See page Нажмите здесь).



4. FILL RESERVOIR WITH BRAKE FLUID



  1. Залейте тормозную жидкость в бачок.

    Тормозная жидкость:
    SAE J1703 или FMVSS № 116 DOT 3

    Обозначения на рисунке 
    *A
    		Для моделей с левосторонним рулевым управлением (для моделей с 2TR-FE, 5L-E)
    *B
    		Для моделей с левосторонним рулевым управлением (с 1GR-FE, 1KD-FTV)
    *C
    		Для моделей с правосторонним рулевым управлением



5. BLEED CLUTCH LINE

  1. Снимите колпачок прокачного штуцера рабочего цилиндра сцепления.

  1. Подсоедините виниловую трубку к прокачному штуцеру (штуцеру для удаления воздуха).

  1. Несколько раз нажмите педаль сцепления, а затем ослабьте прокачной штуцер при нажатой педали.

  1. Когда жидкость прекратит вытекать, заверните прокачной штуцер и отпустите педаль сцепления.

  1. Повторяйте 2 описанных выше операции до тех пор, пока из жидкости не будет удален весь воздух.

  1. Затяните прокачной штуцер.

    Момент затяжки:
    11 N*m{ 110 kgf*cm , 8 ft.*lbf }

  1. Установите колпачок прокачного штуцера.

  1. Проверьте, чтобы из трубопровода системы сцепления был удален весь воздух.



6. CHECK FLUID LEVEL IN RESERVOIR

  1. Проверьте уровень жидкости.
    Если уровень тормозной жидкости мал, проверьте наличие утечек и тормозные колодки дискового тормоза. При необходимости долейте тормозную жидкость в бачок после замены или ремонта.

    Тормозная жидкость:
    SAE J1703 или FMVSS № 116 DOT 3



7. INSPECT FOR BRAKE FLUID LEAK FROM CLUTCH LINE
