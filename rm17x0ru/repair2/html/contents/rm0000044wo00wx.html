ПРОВЕРЬТЕ DTC (С ПОМОЩЬЮ ПОРТАТИВНОГО ДИАГНОСТИЧЕСКОГО ПРИБОРА)

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Установите замок зажигания в положение ON (ВКЛ).

  3. Включите портативный диагностический прибор.

  4. Войдите в следующие меню: Body / Navigation System / DTC.

  5. Проверьте и запишите коды DTC и данные фиксированного набора параметров.

  6. Обратитесь к описанию кодов DTC (Нажмите здесь).

    ПРИМЕЧАНИЕ:
    Система навигации выдает коды DTC для следующих систем. Если выводятся какие-либо коды DTC помимо кодов, указанных в таблице диагностических кодов неисправностей для системы навигации, обратитесь к таблицам диагностических кодов неисправностей для соответствующей системы.

УДАЛИТЕ DTC (С ПОМОЩЬЮ ПОРТАТИВНОГО ДИАГНОСТИЧЕСКОГО ПРИБОРА)

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Установите замок зажигания в положение ON (ВКЛ).

  3. Включите портативный диагностический прибор.

  4. Войдите в следующие меню: Body / Navigation System / DTC.

  5. Удалите DTC.


ЗАПУСК РЕЖИМА ДИАГНОСТИКИ

УКАЗАНИЕ:
  1. Экраны на рисунках могут отличаться от экранов, отображаемых в конкретном автомобиле, из-за различия настроек и дополнительного оборудования. В связи с этим, некоторые области не могут быть отображены в точности так, как вы их видите на своем дисплее.
  2. В целях поиска неисправности, когда система не может перейти в режим диагностики, проверьте DTC с помощью портативного диагностического прибора.
  3. После включения зажигания до запуска режима диагностики убедитесь в том, что карта отображается. В противном случае вы не сможете проверить некоторые параметры.

  1. Есть 2 способа запуска режима диагностики. Войдите в режим с помощью одного из следующих методов.



  2. Метод 1

    1. Запустите двигатель.




    2. Удерживая выключатель "INFO TEL" нажатым, управляйте переключателем освещения: выкл → вкл → выкл → вкл → выкл → вкл → выкл.

    3. Запускается режим диагностики, отображается экран "System Check Mode" (режим проверки системы). Проверка работы запускается автоматически, результаты отображаются.



  3. Метод 2

    1. Запустите двигатель.




    2. Нажмите на выключатель "DISP".




    3. Находясь в окне регулировки качества дисплея, коснитесь углов экрана в следующем порядке: верхний левый → нижний левый → верхний левый → нижний левый → верхний левый → нижний левый.
      Обозначения на рисунке 
      *a
      		Левый верхний
      *b
      		Левый нижний

    4. Запускается режим диагностики, отображается экран "System Check Mode" (режим проверки системы).


ЗАВЕРШЕНИЕ РЕЖИМА ДИАГНОСТИКИ

  1. Выключите зажигание.


ДИАГНОСТИКА НЕИСПРАВНОСТЕЙ



  1. Нажмите кнопку "Failure Diagnosis" (диагностика неисправностей) на экране "Service Menu" (меню обслуживания).




  2. Отображается экран "Failure Diagnosis" (диагностика неисправностей).


ПРОВЕРКА СИСТЕМЫ



  1. Нажмите кнопку "System Check" (проверка системы) на экране "Failure Diagnosis" (диагностика неисправностей).




  2. Отображается экран "System Check Mode" (режим проверки системы).


ПРОВЕРЬТЕ DTC (С ПОМОЩЬЮ ЭКРАНА РЕЖИМА ПРОВЕРКИ СИСТЕМЫ)


  1. Проверьте результаты проверки системы.




    1. Если после проверки отображается результат "EXCH" или "CHEK", прикоснитесь к экрану в этом месте для просмотра содержимого.

      УКАЗАНИЕ:
      Если все результаты "OK", это означает, что коды DTC не зарегистрированы.




    2. Просмотрите результаты на экране "Unit Check Mode" (режим проверки устройства) и запишите их.




    3. Для просмотра результатов проверки другого устройства нажмите кнопку "Back", чтобы вернуться на экран "System Check Mode" (режим проверки системы).

      УКАЗАНИЕ:
      Чтобы проверить другие параметры, повторите эту процедуру, начиная с первого шага.


УДАЛЕНИЕ / ПОВТОРНАЯ ПРОВЕРКА DTC (УДАЛЕНИЕ С ПОМОЩЬЮ ЭКРАНА РЕЖИМА ПРОВЕРКИ СИСТЕМЫ)



  1. Удалите DTC.

    1. Нажмите и удерживайте выключатель "Memory CLR" (сброс памяти) в течение 3 секунд.

    2. Убедитесь, что результаты проверки удалены.

      УКАЗАНИЕ:
      1. Чтобы удалить DTC для определенного устройства, используйте экран "Unit Check Mode".
      2. При удалении DTC с помощью экрана "Unit Check Mode" нажмите переключатель "Code CLR" на 3 с.




  2. Повторная проверка

    1. Нажмите кнопку "Recheck".




    2. Убедитесь в том, что при выводе всех результатов проверки отображается "OK". Если для результата не отображается "OK", произведите поиск и устранение неисправностей еще раз.

      УКАЗАНИЕ:
      Если код DTC удален с экрана "Unit Check Mode", нажмите переключатель "Back", чтобы вернуться на экран "System Check Mode", и выполните данную операцию.
