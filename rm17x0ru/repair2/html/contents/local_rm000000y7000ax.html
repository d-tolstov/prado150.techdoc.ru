DTC    19 (1)    Accelerator Position Sensor Circuit Malfunction


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

When using intelligent tester:

READ VALUE USING INTELLIGENT TESTER (ACCELERATOR POSITION)

READ OUTPUT DTC (CHECK IF ACCELERATOR PEDAL POSITION SENSOR DTC IS OUTPUT AGAIN)

CHECK HARNESS AND CONNECTOR (ECM - ACCELERATOR PEDAL POSITION SENSOR)

CHECK ECM (VC, VCPA VOLTAGE)

REPLACE ACCELERATOR PEDAL ROD ASSEMBLY

READ OUTPUT DTC (CHECK IF ACCELERATOR PEDAL POSITION SENSOR DTC IS OUTPUT AGAIN)

When not using intelligent tester:

CHECK HARNESS AND CONNECTOR (ECM - ACCELERATOR PEDAL POSITION SENSOR)

CHECK ECM (VC, VCPA VOLTAGE)

REPLACE ACCELERATOR PEDAL ROD ASSEMBLY

READ OUTPUT DTC (CHECK IF ACCELERATOR PEDAL POSITION SENSOR DTC IS OUTPUT AGAIN)
