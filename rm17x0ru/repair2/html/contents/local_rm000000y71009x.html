DTC    19 (2)    Accelerator Position Sensor Range / Performance Problem


DESCRIPTION

WIRING DIAGRAM

INSPECTION PROCEDURE

When using intelligent tester:

READ VALUE USING INTELLIGENT TESTER (ACCELERATOR POSITION)

CHECK HARNESS AND CONNECTOR (ECM - ACCELERATOR PEDAL POSITION SENSOR)

When not using intelligent tester:

CHECK ECM (VPA, VPA2 VOLTAGE)

CHECK HARNESS AND CONNECTOR (ECM - ACCELERATOR PEDAL POSITION SENSOR)
