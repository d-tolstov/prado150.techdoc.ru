ОПИСАНИЕ

После замены ЭБУ помощи при движении новый ЭБУ считывает информацию о технических параметрах для страны эксплуатации автомобиля из главного ЭБУ кузова (бортового ЭБУ сети мультиплексной связи). Если ЭБУ помощи при движении не может сохранить информацию о технических параметрах для страны эксплуатации, сохраняется DTC C1A02.

Код DTC
Условие обнаружения DTC
Неисправный участок
C1A02
Когда замок зажигания находится в положении ON (ВКЛ), и действует предаварийная система безопасности, ЭБУ помощи при движении не может сохранить информацию о технических параметрах для страны эксплуатации.
  1. Главный ЭБУ кузова (бортовой ЭБУ сети мультиплексной связи)
  2. ЭБУ помощи при движении
