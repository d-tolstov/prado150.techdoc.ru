1. INSTALL FRONT EXHAUST PIPE ASSEMBLY

  1. Install a new gasket and the front exhaust pipe to the exhaust manifold with 3 new nuts.

    Момент затяжки:
    54 N*m{ 554 kgf*cm , 40 ft.*lbf }

  1. Install the No. 1 exhaust pipe support bracket with the 2 bolts.

    Момент затяжки:
    71 N*m{ 724 kgf*cm , 52 ft.*lbf }

  1. Install the clamp with the bolt.

    Момент затяжки:
    19 N*m{ 194 kgf*cm , 14 ft.*lbf }



2. INSTALL CENTER EXHAUST PIPE ASSEMBLY



  1. Using a vernier caliper, measure the free length of the compression spring.

    Minimum free length:
    43 mm (1.693 in.)

    1. If the free length is less than the minimum, replace the compression spring.




  1. Using a plastic-faced hammer and wooden block, tap on a new gasket until its surface is flush with the front exhaust pipe.
    Text in Illustration 
    *1
    		Gasket
    *2
    		Wooden Block

    ПРИМЕЧАНИЕ:
    1. Be sure to install the gasket so that it is facing in the correct direction.
    2. Do not reuse the gasket.
    3. Do not damage the gasket.
    4. When connecting the exhaust pipe, do not push on the gasket with the exhaust pipe.

  1. Connect the center exhaust pipe to the 4 exhaust pipe supports.

  1. Install the center exhaust pipe with the 2 compression springs and 2 bolts.

    Момент затяжки:
    43 N*m{ 438 kgf*cm , 32 ft.*lbf }



3. INSTALL TAILPIPE ASSEMBLY

  1. Connect the tailpipe to the exhaust pipe support.

  1. Install a new gasket to the center exhaust pipe.

  1. Connect the tailpipe to the center exhaust pipe.

  1. Install the 2 bolts.

    Момент затяжки:
    48 N*m{ 489 kgf*cm , 35 ft.*lbf }



4. INSPECT FOR EXHAUST GAS LEAK
  1. If gas is leaking, tighten the areas necessary to stop the leak. Replace damaged parts as necessary.
