1. INSTALL INJECTION PUMP ASSEMBLY

  1. Temporarily install the injection pump to the timing gear case with the 2 nuts.

  1. Temporarily install the injection pump stay to the injection pump rear end with the 3 bolts.




  1. Rotate the pump body to align the marks on the pump flange and timing gear case.

  1. Tighten the 2 nuts.

    Момент затяжки:
    21 N*m{ 209 kgf*cm , 15 ft.*lbf }

  1. Tighten the 3 bolts.

    Момент затяжки:
    26 N*m{ 265 kgf*cm , 19 ft.*lbf }




  1. Connect the 3 fuel hoses.

  1. Connect the 5 connectors and attach the wire harness clamp.



2. INSTALL INJECTION PUMP DRIVE PULLEY



  1. Using SST, install the injection pump drive pulley with the nut.

    SST
    09213-14010   (91651-60865)
    09330-00021  

    Момент затяжки:
    64 N*m{ 650 kgf*cm , 47 ft.*lbf }



3. INSTALL INJECTION PIPE SET

  1. Install the 2 lower clamps to the intake manifold.




  1. Install the 4 injection pipes.

    Момент затяжки:
    25 N*m{ 250 kgf*cm , 18 ft.*lbf }

    Text in Illustration 
    *a
    		for Injection Nozzle Side
    *b
    		for Injection Pump Side

    ПРИМЕЧАНИЕ:
    Use the formula to calculate special torque values for situations where a union nut wrench is combined with a torque wrench (See page Нажмите здесь).

  1. Install the 2 upper pipe clamps with the 2 nuts.

    Момент затяжки:
    5.0 N*m{ 51 kgf*cm , 44 in.*lbf }



4. INSTALL DIESEL THROTTLE BODY

  1. Install the diesel throttle body (See page Нажмите здесь).



5. INSTALL TIMING BELT

  1. Install the timing belt (See page Нажмите здесь).



6. BLEED INJECTION PIPE

  1. Move the hand pump on the upper part of the fuel filter up and down and fill the injection pump and fuel system with fuel.

  1. Loosen one of the union nuts (on the nozzle side).

  1. Crank the engine until fuel comes out from the union nut connection (on the nozzle side).

  1. Tighten the union nut.

    Момент затяжки:
    25 N*m{ 250 kgf*cm , 18 ft.*lbf }

    ПРИМЕЧАНИЕ:
    Use the formula to calculate special torque values for situations where a union nut wrench is combined with a torque wrench (See page Нажмите здесь).

  1. Perform the procedures above for each injection pipe.
