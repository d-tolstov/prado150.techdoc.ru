СЧИТАЙТЕ ДАННЫЕ В РЕЖИМЕ DATA LIST

УКАЗАНИЕ:
В режиме Data List на дисплей портативного диагностического прибора можно вывести данные о состоянии переключателей, датчиков, приводов и пр. без снятия деталей с автомобиля. Такая внешняя проверка может оказаться очень полезной, поскольку позволяет выявлять нерегулярно возникающие состояния и сигналы, не нарушая монтажа деталей и электропроводки. Считывание информации в режиме Data List на первом этапе поиска неисправностей позволяет сократить время, затрачиваемое на диагностику.

ПРИМЕЧАНИЕ:
Значения, указанные в колонке "Нормальное состояние" приведенной ниже таблицы, являются справочными. Принимая решение об исправности той или иной детали, не ориентируйтесь исключительно на них.

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Включите зажигание (IG).

  3. Включите портативный диагностический прибор.

  4. Войдите в следующие меню: Body / Entry&Start / Data List.

  5. В соответствии с указаниями на дисплее портативного диагностического прибора считайте данные в режиме Data List.

    Entry&Start (посадка и запуск) 
    Информация на дисплее прибора
    		Измеряемая величина / диапазон измерения
    		Нормальное состояние
    		Замечание по диагностике
    Ignition Switch
    		Сигнал включения зажигания (IG) / ON (ВКЛ) или OFF (ВЫКЛ)
    		ON (ВКЛ): Зажигание включено (IG)
    OFF (ВЫКЛ): Зажигание выключено
    		-
    # Codes
    		Количество кодов неисправностей / мин.: 0, макс.: 255
    		Отображается количество зарегистрированных кодов DTC
    		-
    Immobilizer when IG=ON
    		Состояние системы иммобилайзера двигателя при включенном зажигании (IG) / Set или Unset
    		SET: Система иммобилайзера двигателя включена при включенном зажигании (IG)
    UNSET: Система иммобилайзера двигателя выключена при включенном зажигании (IG)
    		-
    Immobiliser
    		Состояние системы иммобилайзера двигателя / SET (ВКЛ) или UNSET (ВЫКЛ)
    		Set (установлено): ключ не находится в салоне
    Unset (не установлено): ключ находится в салоне
    		-
    Master Key
    		Сигнал кода главного ключа / Match или No Match
    		Match (Соответствует): передан верный код главного ключа
    No Match (Не соответствует): передан неверный код главного ключа
    		-
    Sub Key
    		Сигнал кода дополнительного (главного) ключа / Match или No Match
    		Match (Соответствует): передан верный код дополнительного ключа (главного ключа)
    No Match (Не соответствует): передан неверный код дополнительного ключа (главного ключа)
    		-
    BCC Malfunction
    		Сигнал электронной микросхемы ключа / OK или NG
    		ОК: отправлены верные данные
    NG: отправлены неверные данные
    		-
    Abnormal Status
    		Данные микросхемы электронного ключа зажигания / OK или NG
    		ОК: Данные OK
    NG: Ошибка данных
    		-
    Different Encrypt Code
    		Сигнал электронной микросхемы ключа / OK или NG
    		ОК: отправлены верные данные
    NG: отправлены неверные данные
    		-
    Different Serial Number
    		Сигнал электронной микросхемы ключа / OK или NG
    		ОК: отправлены верные данные
    NG: отправлены неверные данные
    		-
    Frame Error
    		Сигнал электронной микросхемы ключа / OK или NG
    		ОК: отправлены верные данные
    NG: отправлены неверные данные
    		-
    Response
    		Сигнал электронной микросхемы ключа / OK или NG
    		ОК: отправлены верные данные
    NG: отправлены неверные данные
    		-
    Wireless Starter Com ID
    		Идентификационный код обмена данными для дистанционного пуска двигателя / No Regd или Regd
    		No Regd: Идентификационный код обмена данными для дистанционного пуска двигателя не зарегистрирован
    Regd: Идентификационный код обмена данными для дистанционного пуска двигателя зарегистрирован
    		-
    Wireless C Code
    		Код C передатчика / No Regd или Regd
    		No Regd: Код C передатчика не зарегистрирован
    Regd: Код C передатчика зарегистрирован
    		-
    ID-BOX Sleep Condition
    		Состояние блока идентификационного кода / YES или NO
    		Yes: блок идентификационного кода находится в режиме ожидания
    No: блок идентификационного кода не находится в режиме ожидания
    		-
    ID-BOX Start Condition
    		Состояние блока идентификационного кода / YES или NO
    		Yes: блок идентификационного кода передает сигнал вывода из режима ожидания
    No: блок идентификационного кода не передает сигнал вывода из режима ожидания
    		-
    Engine Start Request
    		Режим запроса запуска блока идентификационного кода / OK или NG
    		ОК: сигнал запроса запуска получен (последний)
    NG: сигнал запроса запуска не получен (последний)
    		-
    3bit Code Request
    		Результат запроса 3-битного кода / OK или NG
    		ОК: Сигнал результата запроса 3-битного кода получен
    NG: Сигнал результата запроса 3-битного кода не получен

    S Code Check
    		Результат проверки кода S / OK или NG
    		OK: проверка подтвердила неисправность
    NG: проверка не пройдена
    		-
    L Code Check
    		Результат проверки кода L / OK или NG
    		OK: проверка подтвердила неисправность
    NG: проверка не пройдена
    		-
    Unlock Request Receive
    		Состояние приема команды разблокировки рулевого управления / OK или NG
    		ОК: команда разблокировки рулевого управления получена
    NG: команда разблокировки рулевого управления не получена
    		-
    Lock Request Receive
    		Индикация приема команды блокировки рулевого управления / OK или NG
    		ОК: сигнал запроса блокировки рулевого управления получен
    NG: запрос блокировки рулевого управления не получен
    		-
    S Code Check (Past)
    		Результат проверки кода S (последний) / OK или NG
    		ОК: проверка пройдена (последняя)
    NG: проверка не пройдена (последняя)
    		-
    L Code Check (Past)
    		Результат проверки кода L (последний) / OK или NG
    		ОК: проверка пройдена (последняя)
    NG: проверка не пройдена (последняя)
    		-
    EFI Code Receive
    		Код SFI*1 или ECD*2 получен (если зарегистрирован DTC) / OK или NG
    		ОК: код SFI принят
    NG: код SFI не принят
    		-
    EFI communication
    		Обмен данными с SFI*1 или ECD*2 / OK or NG
    		OK: Обмен данными с SFI*1 или ECD*2 осуществляется надлежащим образом
    NG: Обмен данными с SFI*1 или ECD*2 осуществляется ненадлежащим образом
    		-
    R Code Check
    		Результат проверки кода R / OK или NG
    		OK: проверка подтвердила неисправность
    NG: проверка не пройдена

    1. *1: для моделей с 1GR-FE
    2. *2: для моделей с 1KD-FTV


ВЫПОЛНИТЕ ПРОВЕРКУ В РЕЖИМЕ ACTIVE TEST

УКАЗАНИЕ:
Выполнение активной диагностики с использованием портативного диагностического прибора дает возможность управлять реле, электровакуумными клапанами, приводами и т.д. без снятия деталей с автомобиля. Такая внешняя функциональная проверка может оказаться очень полезной, поскольку позволяет выявлять эпизодические неисправности, не нарушая монтаж деталей и электропроводки. Выполнение активной диагностики на начальном этапе поиска неисправностей позволяет сократить время, затрачиваемое на диагностику. В режиме Active Test на дисплей диагностического прибора можно вывести список параметров Data list.

  1. Подсоедините портативный диагностический прибор к DLC3.

  2. Включите зажигание (IG).

  3. Включите портативный диагностический прибор.

  4. Войдите в следующие меню: Body / Entry&Start / Active Test.

  5. В соответствии с указаниями на дисплее портативного диагностического прибора выполните диагностику в режиме Active Test.

    Entry&Start (посадка и запуск) 
    Информация на дисплее прибора
    		Испытываемое устройство
    		Диапазон регулирования
    		Замечание по диагностике
    Immobiliser Indicator
    		Контрольная лампа безопасности
    		ON (ВКЛ) / OFF (ВЫКЛ)
    		Испытание возможно при выполнении следующих условий:
    1. Ключ находится в салоне.
    2. Выключатель зажигания установлен в состояние ON (ВКЛ) (IG).
