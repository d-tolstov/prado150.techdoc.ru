DTC    C1611    Неисправность ЭБУ

для подготовки Нажмите здесь


ОПИСАНИЕ

Данный DTC сохраняется, если ЭБУ системы помощи при парковке в результате самодиагностики обнаруживает внутреннюю неисправность.

УКАЗАНИЕ:
Если при включении ЭБУ системы помощи при парковке не может считать сохраненную информацию, он регистрирует внутреннюю неисправность и сохраняет данный DTC.


Код DTC
Условие обнаружения DTC
Неисправный участок
C1611
  1. EEPROM Status (Initial read)
  2. AISIC Output Status
ЭБУ системы помощи при парковке

ПОСЛЕДОВАТЕЛЬНОСТЬ ПРОВЕРКИ

1.СНИМИТЕ ПОКАЗАНИЯ ПОРТАТИВНОГО ДИАГНОСТИЧЕСКОГО ПРИБОРА

  1. Подсоедините портативный диагностический прибор к DLC3.

  1. Установите замок зажигания в положение ON (ВКЛ).

  1. Включите портативный диагностический прибор.

  1. Войдите в следующие меню: Chassis / Parking Assist Monitor System / Data List.

  1. Проверьте в режиме Data List следующие параметры.

    Система помощи при парковке 
    Информация на дисплее прибора
    		Измеряемая величина / диапазон измерения
    		Нормальное состояние
    		Замечание по диагностике
    EEPROM Status (Initial read)
    		Информация ЭБУ системы помощи при парковке / OK или NG
    		ОК: ЭБУ системы помощи при парковке исправен
    NG: ЭБУ системы помощи при парковке неисправен
    		-
    AISIC Output Status
    		Состояние выходного сигнала изображения камеры / OK или NG
    		ОК: нормальное состояние выходного сигнала изображения камеры
    NG: ненормальное состояние выходного сигнала изображения камеры
    		-
    Результат 
    Результат
    		Следующий шаг
    Для обоих параметров отображается "OK"
    		А
    Для параметра EEPROM Status (Initial read) отображается "NG"
    		А
    Для параметра AISIC Output Status отображается NG (для моделей с левосторонним рулевым управлением)
    		B
    Для параметра AISIC Output Status отображается NG (для моделей с правосторонним рулевым управлением)
    		C
