DTC    C168B    Сбой передачи данных AVC-LAN

DTC    C168C    Истечение времени ожидания команды AVC-LAN

для подготовки Нажмите здесь


ОПИСАНИЕ

Данный DTC сохраняется, если ЭБУ системы помощи при парковке в результате самодиагностики обнаруживает неисправность системы передачи данных AVC-LAN.

УКАЗАНИЕ:
Причиной регистрации данного DTC может быть состояние аккумуляторной батареи или форма напряжения питания при запуске двигателя, хотя в системе передачи данных AVC-LAN нет проблем.


Код DTC
Условие обнаружения DTC
Неисправный участок
C168B
  1. Сбой передачи данных AVC-LAN: ведущее устройство не обнаружено
  2. Сбой передачи данных AVC-LAN: сброс ведущего устройства
  3. Сбой передачи данных AVC-LAN: неисправность ведущего устройства
  4. Сбой передачи данных AVC-LAN: ненормальная проверка соединения
  5. Сбой передачи данных AVC-LAN: передача не завершена
  6. Сбой передачи данных AVC-LAN: уведомление о ненормальном завершении передачи
  7. Сбой передачи данных AVC-LAN: передача запроса на регистрацию
  8. Сбой передачи данных AVC-LAN: ненормальный параметр ON/OFF (ВКЛ/ВЫКЛ)
Система передачи данных AVC-LAN
C168C
  1. Истечение времени ожидания команды AVC-LAN: истечение времени ожидания команды уведомления о запросе ON (ВКЛ) на выходе
  2. Истечение времени ожидания команды AVC-LAN: истечение времени ожидания команды уведомления о запросе OFF (ВЫКЛ) на выходе
  3. Истечение времени ожидания команды AVC-LAN: истечение времени ожидания команды уведомления о запросе начала формирования голосового сигнала
  4. Истечение времени ожидания команды AVC-LAN: истечение времени ожидания команды уведомления о запросе окончания формирования голосового сигнала
