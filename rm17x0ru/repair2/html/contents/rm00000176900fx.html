ГИДРОАККУМУЛЯТОР ПРИВОДА СЦЕПЛЕНИЯ (для моделей с левосторонним рулевым управлением) > СНЯТИЕ

для подготовки Нажмите здесь



1. СЛЕЙТЕ ТОРМОЗНУЮ ЖИДКОСТЬ ИЗ ТРУБОПРОВОДА ПРИВОДА СЦЕПЛЕНИЯ


2. СНИМИТЕ ПЕРЕДНИЙ КАРДАННЫЙ ВАЛ В СБОРЕ (для моделей с 1GR-FE)

  1. Снимите передний карданный вал (Нажмите здесь).



3. СНИМИТЕ КОЖУХ КАРТЕРА СЦЕПЛЕНИЯ № 1 (для моделей с 1GR-FE)



  1. Выверните 4 болта и снимите крышку картера.



4. СНИМИТЕ ГИДРОАККУМУЛЯТОР ПРИВОДА СЦЕПЛЕНИЯ В СБОРЕ


  1. Для моделей с 1GR-FE:

    1. Снимите трубопровод от рабочего цилиндра сцепления к гидроаккумулятору (Нажмите здесь).




    2. С помощью разрезной головки отсоедините трубопровод, соединяющий рабочий цилиндр сцепления с гибким шлангом.

      УКАЗАНИЕ:
      Используйте резервуар для сбора жидкости.

    3. Отверните гайку.




    4. Выверните 3 болта и снимите гидроаккумулятор привода сцепления.



  1. За исключением моделей с 1GR-FE:




    1. С помощью разрезной головки отсоедините трубопровод, соединяющий главный цилиндр сцепления с гибким шлангом.

      УКАЗАНИЕ:
      Используйте резервуар для сбора жидкости.




    2. С помощью разрезной головки отсоедините трубопровод между гидроаккумулятором и гибким шлангом.

      УКАЗАНИЕ:
      Используйте резервуар для сбора жидкости.




    3. Отверните 3 гайки и снимите гидроаккумулятор привода сцепления.
