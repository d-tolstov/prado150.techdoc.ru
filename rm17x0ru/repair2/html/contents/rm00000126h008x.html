1. INSTALL NO. 1 TIMING BELT IDLER SUB-ASSEMBLY



  1. Install the No. 1 belt idler with the 3 bolts.

    Момент затяжки:
    for bolt A:
    44 N*m{ 449 kgf*cm , 32 ft.*lbf }
    for bolt B, C:
    19 N*m{ 195 kgf*cm , 14 ft.*lbf }

    УКАЗАНИЕ:


    2. The bolt lengths for bolt A, B and C as follows.
    3. Bolt C is combined with the No. 1 timing belt idler.

    Standard Bolt:
    Item
    		Length
    A
    		76.5 mm (3.01 in.)
    B
    		42.9 mm (1.69 in.)
    C
    		41.3 mm (1.63 in.)



2. SET NO. 1 CYLINDER TO TDC/COMPRESSION



  1. Using the crankshaft pulley bolt, align the groove of the crankshaft pulley with the timing pointer by turning the crankshaft clockwise.
    Text in Illustration 
    *1
    		Timing Mark

    		Turn

    ПРИМЕЧАНИЕ:
    Do not turn the crankshaft pulley counterclockwise.




  1. Set the timing and drive pulleys at each position.

    ПРИМЕЧАНИЕ:
    1. Make sure the engine is cold.
    2. When turning the crankshaft or camshaft, the valve heads will hit against the piston top. Do not turn them more than necessary.



3. INSTALL TIMING BELT



УКАЗАНИЕ:
If reusing the timing belt, align the points marked during removal, and install the timing belt with the arrow pointing in the direction the belt moves when the engine is running.

  1. Remove any oil or water on each pulley, and keep them clean.

  1. Install the timing belt to the crankshaft timing and timing belt idlers.




  1. Using SST, slightly turn the injection pump drive pulley clockwise. Install the timing belt to the pulley, and align the timing marks of the drive pulley and timing belt case.

    SST
    09960-10010   (09962-01000, 09963-01000)
    Text in Illustration 
    *1
    		Timing Mark




  1. Using SST, slightly turn the camshaft timing pulley clockwise. Install the timing belt to the timing pulley, and align the timing marks of the timing pulley and timing belt case.

    SST
    09960-10010   (09962-01000, 09963-01000)
    Text in Illustration 
    *1
    		Timing Mark

  1. Check that the timing belt has tension between the injection pump drive and camshaft timing pulleys.

  1. Install the timing belt to the No. 1 timing belt idler.




  1. Loosen the No. 1 timing belt idler bolt (A), and stretch the timing belt.

  1. Slowly turn the crankshaft pulley.

    ПРИМЕЧАНИЕ:
    Always turn the crankshaft clockwise.

  1. Tighten the No. 1 timing belt idler bolt.

    Момент затяжки:
    44 N*m{ 449 kgf*cm , 33 ft.*lbf }



4. CHECK NO. 1 CYLINDER TO TDC/COMPRESSION

  1. Slowly turn the crankshaft pulley 2 revolutions from TDC to TDC.

    ПРИМЕЧАНИЕ:
    Always turn the crankshaft clockwise.




  1. Check that the timing marks for each pulley align as shown in the illustration.
    If the timing marks do not align, remove the timing belt and reinstall it.



5. INSTALL TIMING BELT GUIDE



  1. Install the timing belt guide with the cup side facing outward.



6. INSTALL TIMING BELT COVER



  1. Install 2 new gaskets to the timing belt cover.
    Text in Illustration 
    *1
    		Gasket




  1. Install the timing belt cover with the 11 bolts and washers.

    Момент затяжки:
    for bolt A:
    8.5 N*m{ 85 kgf*cm , 75 ft.*lbf }
    except bolt A:
    11 N*m{ 107 kgf*cm , 8 ft.*lbf }



7. INSTALL IDLE PULLEY ASSEMBLY

  1. Install the idle pulley bracket with the 2 bolts.

    Момент затяжки:
    29 N*m{ 300 kgf*cm , 22 ft.*lbf }



8. INSTALL CRANKSHAFT PULLEY

  1. Align the key groove of the pulley with the pulley set key, and slide the pulley onto the crankshaft to install it.




  1. Using SST, install the pulley bolt.

12. INSPECT ENGINE IDLE SPEED

  1. Warm up the engine.

  1. When using the intelligent tester:

    1. Connect the intelligent tester to the DLC3.

      Idle speed:
      720 to 820 rpm

      ПРИМЕЧАНИЕ:
      1. Turn all the electrical systems and the A/C off.
      2. When checking the idling speed, the shift lever should be in neutral.

      УКАЗАНИЕ:
      Refer to the intelligent tester operator's manual for further details.




  1. When not using an intelligent tester:

    1. Using SST, connect the tachometer test probe to terminal 9 (TAC) of the DLC3.

    2. SST
      09843-18030  
      Check the idle speed.

      Standard idle speed:
      720 to 820 rpm

      Text in Illustration 
      *a
      		Front View of DLC3

      ПРИМЕЧАНИЕ:
      1. Turn all the electrical systems and the A/C off.
      2. When checking the idling speed, the shift lever should be in neutral.
      3. Confirm the terminal number before connecting them. Connecting the wrong terminal can be damage the engine.



13. INSPECT MAXIMUM ENGINE SPEED

  1. Start the engine.

  1. Fully depress the accelerator pedal.

  1. Check the maximum speed.

    Maximum engine speed:
    4850 to 4950 rpm
