1. ПРОВЕРЬТЕ ГИДРАВЛИЧЕСКИЙ КЛАПАН ИЗМЕНЕНИЯ ФАЗ В СБОРЕ

  1. Подсоедините портативный диагностический прибор к DLC3.

  1. Запустите двигатель.

  1. Включите портативный диагностический прибор.

  1. Проверьте гидравлический клапан изменения фаз распредвала впускных клапанов.

    1. Войдите в следующие меню: Powertrain / Engine / Active Test / Control the VVT System (Bank 1).

    2. С помощью портативного диагностического прибора управляйте гидравлическим клапаном изменения фаз и проверьте частоту вращения коленчатого вала двигателя.

      Нормальное состояние:
      Диапазон регулирования
      		Заданные условия
      OCV ВЫКЛ
      		Нормальная частота вращения коленчатого вала двигателя
      OCV ВКЛ
      		Двигатель глохнет или неравномерно работает на холостом ходу сразу после переключения клапана из выключенного состояния во включенное

      Если результат проверки не соответствует указанному, проверьте гидравлический клапан изменения фаз, жгут проводов и ЕСМ.

    3. Войдите в следующие меню: Powertrain / Engine / Active Test / Control the VVT System (Bank 2).

    4. С помощью портативного диагностического прибора управляйте гидравлическим клапаном изменения фаз и проверьте частоту вращения коленчатого вала двигателя.

      Нормальное состояние:
      Диапазон регулирования
      		Заданные условия
      OCV ВЫКЛ
      		Нормальная частота вращения коленчатого вала двигателя
      OCV ВКЛ
      		Двигатель глохнет или неравномерно работает на холостом ходу сразу после переключения клапана из выключенного состояния во включенное

      Если результат проверки не соответствует указанному, проверьте гидравлический клапан изменения фаз, жгут проводов и ЕСМ.

  1. Проверьте гидравлический клапан изменения фаз распредвала выпускных клапанов.

    1. Войдите в следующие меню: Powertrain / Engine / Active Test / Control the VVT Exhaust Linear (Bank 1).

    2. С помощью портативного диагностического прибора управляйте гидравлическим клапаном изменения фаз и проверьте частоту вращения коленчатого вала двигателя.

      Нормальное состояние:
      Диапазон регулирования
      		Заданные условия
      -100%
      		Нормальная частота вращения коленчатого вала двигателя
      100%
      		Двигатель глохнет или неравномерно работает на холостом ходу сразу после переключения клапана из выключенного состояния (-100%) во включенное (+100%).

      Если результат проверки не соответствует указанному, проверьте гидравлический клапан изменения фаз, жгут проводов и ЕСМ.

    3. Войдите в следующие меню: Powertrain / Engine / Active Test / Control the VVT Exhaust Linear (Bank 2).

    4. С помощью портативного диагностического прибора управляйте гидравлическим клапаном изменения фаз и проверьте частоту вращения коленчатого вала двигателя.

      Нормальное состояние:
      Диапазон регулирования
      		Заданные условия
      -100%
      		Нормальная частота вращения коленчатого вала двигателя
      100%
      		Двигатель глохнет или неравномерно работает на холостом ходу сразу после переключения клапана из выключенного состояния (-100%) во включенное (+100%).

      Если результат проверки не соответствует указанному, проверьте гидравлический клапан изменения фаз, жгут проводов и ЕСМ.
