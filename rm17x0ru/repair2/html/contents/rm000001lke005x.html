1. INSTALL NOZZLE RETAINING NUT



  1. Install the pressure pin, distance piece and nozzle assembly to the nozzle retaining nut.

  1. Install the nozzle holder pressure spring and adjusting shim, and then temporarily install the nozzle holder to the nozzle retaining nut.

  1. Using SST, tighten the nozzle retaining nut.

    SST
    09268-64010   (09268-64020, 09268-64030)

    Момент затяжки:
    47 N*m{ 475 kgf*cm , 34 ft.*lbf }

    ПРИМЕЧАНИЕ:
    Overtightening could cause nozzle deformation and needle adhesion or other defects.

  1. Inspect the holder and nozzle set.



2. INSTALL RING PACKING SETTING NUT

  1. Install a new washer and the ring packing setting nut.
