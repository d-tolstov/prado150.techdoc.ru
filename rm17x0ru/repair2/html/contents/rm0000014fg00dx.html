1. INSTALL GENERATOR ROTOR ASSEMBLY



  1. Using SST and a press, press in the generator rotor.

    SST
    09285-76010  

  1. Place the washer on the generator rotor.



2. INSTALL GENERATOR RECTIFIER END FRAME



  1. Using a 17 mm socket wrench and press, slowly press on the rectifier end frame.
    Text in Illustration 
    *1
    		17 mm Socket Wrench

  1. Install the cord clip and 4 nuts.

    Момент затяжки:
    5.9 N*m{ 60 kgf*cm , 52 in.*lbf }



3. INSTALL GENERATOR HOLDER WITH RECTIFIER



  1. Install the 4 rubber insulators to the lead wires.
    Text in Illustration 

    		Inside

    ПРИМЕЧАНИЕ:
    Be careful to install the rubber insulators in the correct direction.




  1. While pushing down on the rectifier holder, install it with the 4 screws.

    Момент затяжки:
    2.9 N*m{ 30 kgf*cm , 26 in.*lbf }

    Text in Illustration 

    		Push



4. INSTALL GENERATOR REGULATOR ASSEMBLY

  1. Install the regulator with the 3 screws.

    Момент затяжки:
    2.0 N*m{ 20 kgf*cm , 18 in.*lbf }



5. INSTALL GENERATOR BRUSH HOLDER ASSEMBLY

  1. Install the plate seal.




  1. Install the brush holder with the 2 screws.

    Момент затяжки:
    2.0 N*m{ 20 kgf*cm , 18 in.*lbf }

    Text in Illustration 

    		Upper

  1. Install the brush holder cover.



6. INSTALL GENERATOR REAR END COVER



  1. Install the rear end cover and rectifier plate with the bolt and 3 nuts.

    Момент затяжки:
    for nut:
    4.4 N*m{ 45 kgf*cm , 39 in.*lbf }
    for bolt:
    3.9 N*m{ 40 kgf*cm , 35 in.*lbf }

    Text in Illustration 
    *1
    		Rectifier Plate

  1. Install the terminal insulator with the nut.

    Момент затяжки:
    4.1 N*m{ 42 kgf*cm , 36 in.*lbf }



7. INSTALL VACUUM PUMP ASSEMBLY

  1. Align the matchmarks and install the vacuum pump with the 4 bolts.

    Момент затяжки:
    7.8 N*m{ 80 kgf*cm , 69 in.*lbf }
