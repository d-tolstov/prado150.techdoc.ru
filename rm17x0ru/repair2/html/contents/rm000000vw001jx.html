1. INSTALL NO. 2 ECM BRACKET

  1. Install the No. 2 ECM bracket with the 2 screws.

    Момент затяжки:
    3.0 N*m{ 31 kgf*cm , 27 in.*lbf }



2. INSTALL NO. 1 ECM BRACKET

  1. Install the No. 1 ECM bracket with the 2 screws.

    Момент затяжки:
    3.0 N*m{ 31 kgf*cm , 27 in.*lbf }



3. INSTALL WIRE HARNESS CLAMP BRACKET (for LHD)

  1. Install the wire harness clamp bracket with the 2 screws.

    Момент затяжки:
    3.0 N*m{ 31 kgf*cm , 27 in.*lbf }



4. INSTALL ECM

  1. Install the ECM with the bolt and nut.

    Момент затяжки:
    8.0 N*m{ 82 kgf*cm , 71 in.*lbf }

  1. Connect the 4 connectors.



5. INSTALL GLOVE COMPARTMENT DOOR ASSEMBLY

  1. Install the glove compartment door (See page Нажмите здесь).
