ЗАМЕНА КОНТАКТОВ

При замене контакта следует выполнить следующие операции.

ПРИМЕЧАНИЕ:
Алюминиевые провода ремонту не подлежат.
Шаг 1. Определите тип разъема и контакта.

Шаг 2. Отсоедините контакт от разъема.

Шаг 3. Замените контакт.

Шаг 4. Вставьте контакт в разъем.

Шаг 1. Определите тип разъема и контакта.

Определите по рисункам в следующих таблицах, какой разъем требуется по типу контактов: в водозащитном исполнении, не в водозащитном исполнении или комбинированный.

Разъем Описание
Не в водозащитном исполнении Эти разъемы не являются водонепроницаемыми или имеют комбинированный тип контактов.
В водозащитном исполнении Для уплотнения контакта / корпуса разъема используется водонепроницаемый материал (пробка отверстия или уплотнение контакта).
С комбинированным типом контактов В одном разъеме используются контакты различной формы/размера.

1. Контакт с проводом

Определите каталожный номер контакта с проводом, выбрав соответствующий рисунок из приведенных ниже таблиц.
УКАЗАНИЕ:
  • В некоторых случаях для одного рисунка контакта с проводом приведено два каталожных номера, так как несмотря на то, что форма контактов может быть идентична, они могут отличаться наличием позолоты.
Примечание:
Длина провода, соединенного с контактом, составляет около 150 мм.
ПРИМЕЧАНИЕ:
  • Если используется позолоченный контакт, при ремонте обязательно заменяйте его позолоченным контактом.
  • Не используйте штыревой и гнездовой контакты, изготовленные из различных материалов.

2. Корпус разъема

  1. Определите каталожный номер контакта с проводом, соответствующего шагу 1.

  1. Определите каталожный номер корпуса разъема, выбрав его по рисункам в соответствии числом контактов и формой.
УКАЗАНИЕ:
(a) Если не найден разъем, соответствующий требуемому типу, выберите комплект корпусов штыревого и гнездового разъемов, имеющий больше контактов, чем требуется
Для корпуса разъема с комбинированным типом контактов выберите соответствующий корпус разъема для каждого используемого контакта.
Пример:
  • Когда в одном корпусе разъема используется два типа контактов, выберите штыревую и гнездовую части для двух различных корпусов разъемов.
(b) Каталожный номер указан на обратной стороне корпуса разъема. (Так будет и для новых типов разъемов.)

3. Пробка отверстия / уплотнение контакта

  1. Определите по рисункам разъема в водозащитном исполнении каталожный номер контакта с проводом / корпуса разъема в соответствии с шагами 1 и 2.

  1. Определите каталожный номер пробки отверстия / уплотнения контакта по соответствующему рисунку.

4. Втулка

При соединении двух проводов с помощью втулки выберите надлежащую втулку из следующей таблицы.

  1. Когда размер основан на наружном диаметре провода.
